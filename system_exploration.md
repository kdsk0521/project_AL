# 탐사 시스템 설계 (v0.6)

> 상위 문서: game_blueprint.md (코어 시스템 1축: 탐사)

---

## 개요

탐사는 게임의 **재료 유입 파이프라인**이자 스태미나를 가장 많이 소모하는 행동.
던전에 들어가서 채집, 전투, 이벤트를 겪고 귀환하여 재료를 가져온다.

```
탐사 준비 (편성, 장비, 소비재) → 던전 진입 → 층/구역 탐색 → 채집/전투/이벤트 → 귀환
       ↓                                                              ↓
  스태미나 소모                                              재료/로어/경험 획득
```

---

## 설계 원칙

> **"구조는 알지만, 안에서 무슨 일이 일어날지는 모른다."**

- 던전의 큰 틀(층, 구역, 보스)은 고정 → 공략의 재미, 계획 가능
- 내부(적 배치, 이벤트, 채집 포인트, 드랍)는 매번 랜덤 → 반복 탐사에 변화
- 탐사는 단순 파밍이 아닌 **비용과 위험을 수반하는 의사결정** (닼던 참고)
- 탐사 결과가 경영/관계/조합 전체에 파급 → 시스템 간섭의 핵심 공급원

---

## 던전 구조

### 구조 방식: 고정 골격 + 랜덤 내부 (세계수 × 이상한 던전)

#### 고정 요소 (매번 동일)

- **층수**: 1층, 2층, 3층... 무한 확장
- **구역 배치**: 각 층의 구역 구성 (광석 구역, 수풀 구역, 수계 구역 등)
- **보스 위치**: 특정 층(5층, 10층, 15층 등)마다 보스 배치
- **채집 가능 재료 종류**: "3층에 가면 철광석이 나온다"는 고정
- **재료 tier**: 깊은 층일수록 상위 tier 재료
- **환경 특성**: 층마다 속성/특성이 있음 (화산층, 수몰층, 결빙층 등)

#### 랜덤 요소 (매 진입 시 변동)

- **적 배치**: 어떤 적이 어디에 있는지
- **채집 포인트 위치**: 어디서 캘 수 있는지
- **드랍 내용**: 같은 적/포인트여도 나오는 구체 아이템이 다름
- **이벤트**: 층 내 랜덤 이벤트 발생 (이머전트 이벤트 풀에서 추첨)
- **숨겨진 구역**: 일정 확률로 출현하는 숨겨진 방/통로
- **유닛 조우**: 특정 조건에서 야생/방랑 유닛이 출현 (관계 시스템 연결)

### 층 구조 예시 (초안)

```
[1~5층: 초급 구간]
  - 기초 재료 (tier 1)
  - 약한 적
  - 5층: 첫 보스
  - 기본 채집 도구로 접근 가능

[6~15층: 중급 구간]
  - 중급 재료 (tier 2)
  - 환경 특성 등장 (화산, 수몰 등)
  - 10층, 15층: 보스
  - 상위 채집 도구 필요
  - 숨겨진 구역 출현 시작

[16~30층: 상급 구간]
  - 상급 재료 (tier 3)
  - 복합 환경 (화산+독기 등)
  - 특수 이벤트 빈도 증가
  - 희귀 유닛 조우 가능

[31층~: 심층]
  - 최상위 재료 (tier 4+)
  - 노드 분기 관련 구역 출현
  - 파편 로어 밀도 최대
  - 무한 확장 (층수 스케일링)
```

### 무한 확장 구조

- 층수에 상한 없음. 깊이 갈수록 난이도/보상이 스케일링
- 특정 층마다 마일스톤 (최초 돌파 시 해금 보상)
- 일정 구간마다 새로운 환경 특성, 재료 tier, 이벤트 풀 추가
- **"아직 안 가본 층이 있다"가 탐사 동기**

## 탐사 흐름

### 1. 탐사 준비

```
- 파티 편성: 플레이어 캐릭터(필수) + 동행 유닛 선택
- 장비 확인: 플레이어 및 유닛 장비 상태
- 소비재 준비: 물약, 음식, 폭탄 등 (인벤토리에서 가져감)
- 도구 확인: 채집 도구 등급 → 채집 가능 재료 범위 결정
- 목표층 선택: 어디까지 갈 것인가 (이전 도달 최고층까지 직행 가능? — 미결정)
```

### 2. 던전 내 행동

층에 진입하면 탐색 시작. 각 층에서 가능한 행동:

- **이동**: 다음 구역/층으로 이동
- **채집**: 채집 포인트에서 재료 획득 (도구 등급에 따라 가용 범위)
- **전투**: 적 조우 시 전투 진입
- **이벤트**: 랜덤/조건부 이벤트 발생 시 대응
- **탐색**: 숨겨진 구역/아이템 발견 시도
- **귀환**: 현재 시점에서 던전을 나감

### 3. 귀환 판단

탐사의 핵심 의사결정 중 하나.

> **"더 깊이 가면 더 좋은 재료가 있지만, 유닛이 다치고 소비재가 줄고 있다."**

- 언제든 귀환 가능 (귀환 아이템/스킬 필요 여부 — 미결정)
- 전멸 시 강제 귀환 + 채집물 일부 소실? (미결정)
- 귀환 시 채집한 재료, 발견한 로어, 획득한 경험이 확정
- 유닛 부상은 귀환 후에도 유지 → 경영/관계에 파급

## 전투 시스템

### 방식: 자동전투 + 최소 개입 (백팩 히어로 참고)

> **"전투의 재미는 싸우는 도중이 아니라, 싸우기 전에 어떻게 세팅했느냐에 있다."**

전투 자체는 자동으로 진행된다. 유닛은 보유 트레잇(스킬)과 장비에 따라 자동 행동.
플레이어가 개입하는 지점은 **전투 전 준비**와 **전투 중 최소 판단** 두 곳뿐.

### 전투 전: 준비가 곧 전투

전투의 승패를 결정하는 것은 전투 전에 이미 정해져 있다.

- **파티 편성**: 누구를 데려가느냐 (트레잇 시너지, 속성 상성)
- **장비 세팅**: 전투 수치는 장비에 의존 → 어떤 장비를 장착했느냐가 핵심
- **소비재 준비**: 물약, 음식, 폭탄 → 전투 중 플레이어가 사용할 자원
- **트레잇 조합**: 유닛 트레잇 간 시너지/상충 → 파티 전체의 전투 양상을 결정

### 전투 중: 유닛 자동 + 플레이어 최소 개입

```
[전투 시작]
       ↓
[매 턴 자동 진행]
  - 유닛: 보유 스킬 트레잇에 따라 자동 행동
  - 행동 패턴은 성격 트레잇에도 영향받음
    (공격적 → 공격 우선, 소심 → 방어 우선 등)
  - 플레이어 캐릭터도 자동 행동 (세팅에 따라)
       ↓
[플레이어 개입 가능 (선택적)]
  - 소비재 사용: 물약, 폭탄 등 소모품 투입
  - 철수 판단: 이 전투에서 빠진다 (도주)
  - 그 외 전투 행동은 없음 — 유닛이 알아서 싸움
       ↓
[전투 종료]
  - 승리: 적 드랍 획득, 경험치 배분
  - 패배: 유닛/플레이어 부상, 강제 귀환 판정
```

### 전투 후: 계속할 것인가

전투가 끝나면 탐사 계속 여부를 판단.

- HP/소비재 잔량 확인
- 부상 유닛 확인
- "이 상태로 다음 층에 가도 되는가?" → 귀환 판단

### 전투 결과를 결정하는 요소

| 요소 | 영향 | 플레이어가 제어하는 타이밍 |
|------|------|------------------------|
| 파티 편성 | 누가 싸우느냐 | 탐사 준비 |
| 장비 | 전투 수치 (공/방/속) | 조합/제작 → 장착 |
| 스킬 트레잇 | 전투 중 자동 행동 | 육성/합체 |
| 성격 트레잇 | 행동 패턴/우선순위 | 육성 |
| 소비재 | 전투 중 유일한 개입 수단 | 조합/제작 → 탐사 준비 |
| 속성 상성 | 유닛 vs 적 타입 매칭 | 파티 편성 |

→ **전투에서 직접 조작하는 건 거의 없지만, 전투 결과를 좌우하는 모든 것은 플레이어가 사전에 결정한 것.**

### 성격 트레잇과 전투 행동

유닛의 성격 트레잇이 자동전투 행동 패턴에 영향을 준다.
같은 스킬을 가져도 성격에 따라 다른 행동을 한다.

```
[공격적] → 공격 우선, 강적에게도 돌진
[소심]   → 방어/회피 우선, HP 낮으면 뒤로 빠짐
[냉정]   → 가장 효율적인 대상 선택, 상성 계산
[호기심] → 랜덤 행동 확률 증가, 예측 불가능
[순종적] → 안정적 패턴, 파티 협동기 발동률 증가
```

이것이 "같은 유닛이라도 어떻게 키웠느냐에 따라 전투에서도 다르게 행동한다"의 구현.

### 플레이어 캐릭터

- 탐사 파티의 **필수 멤버** (빠질 수 없음)
- 자체 스탯, 장비, 스킬 보유
- 전투 중에는 유닛과 마찬가지로 자동 행동
- 성장 가능 (레벨/경험치 — 상세 미결정)
- 유닛과의 차이점:
  - **판매 불가** (플레이어 자신이므로)
  - **상실 불가** (부상은 가능, 사망은 없음)
  - 합체 소재로 사용 불가
- 플레이어 부상 시:
  - 일정 기간 탐사 불가 → 그 동안 가공/조합/판매/육성만 가능
  - 치료 아이템으로 회복 단축 가능
  - **플레이어 부상 = 탐사 수입 완전 차단** → 가장 큰 리스크
- **전투에서 게임오버는 없다**: 전멸/플레이어 쓰러짐 시 강제 귀환 + 행동불가 페널티
  - 행동불가 기간(일 수)은 부상 정도에 따라 차등 (밸런싱)
  - 행동불가 중에도 고정 지출(세금, 유지비)은 계속 나감 → 실질적 경제 패널티
  - **게임오버는 경영 쪽(파산 엔딩 0-X)에서만 발생** → 진짜 리스크는 경영이 무너지는 것

### 전투 관련 미결정

- [ ] 유닛 스킬/속성 상성 시스템
- [ ] 부상 단계 (경상/중상?)
- [ ] 부상 회복 방식 (시간? 아이템? 자금?)
- [ ] 보스전 특수 규칙 (자동전투와 동일? 추가 개입 가능?)
- [ ] 파티 시너지 구조 (특정 트레잇 조합이 파티 버프를 주는가)
- [ ] 플레이어 캐릭터 성장 방식 (레벨? 스킬 트리? 장비 의존?)
- [ ] 전투 속도/연출 (텍스트 로그 형식, 요약 형식 등)
- [ ] 도주 성공률/실패 리스크

## 채집 시스템

### 핵심 구조

던전 내 채집 포인트에서 재료를 획득. **채집 도구 등급**이 접근 가능 범위를 결정.

```
[채집 판정]
채집 포인트 발견 → 도구 등급 확인
  → 도구 등급 충분: 채집 성공 → 재료 획득 (드랍 테이블에서)
  → 도구 등급 부족: "더 좋은 도구가 필요하다" → 채집 불가
```

### 채집 포인트 유형

| 유형 | 필요 도구 | 산출 재료 | 예시 |
|------|----------|----------|------|
| 광맥 | 곡괭이 | 광석류 | 철광석, 미스릴 원석 |
| 수풀 | 채집 낫 | 식물류 | 약초, 마나초, 독풀 |
| 수계 | 낚시대 | 수산/액체류 | 물고기, 마력수, 심층수 |
| 특수 | 전용 도구 | 희귀 재료 | 결정, 화석, 마석 등 |

- 같은 유형이라도 층에 따라 나오는 재료 tier가 다름
- 채집 시 스태미나 소모 (소량)

## 탐사 비용 구조

탐사는 **가장 스태미나를 많이 소모하는 행동**이자 소비재를 소진하는 행동.

### 스태미나 소모

```
탐사 1회 스태미나 = 기본 비용 + (목표 층수 × 층당 추가 비용)
```

- 깊은 층을 갈수록 스태미나 소모가 큼 → 하루에 깊은 탐사와 다른 행동을 병행하기 어려움
- 얕은 층 파밍은 스태미나 저비용 → 빠르게 기초 재료 확보 가능
- 탐사 후 남은 스태미나로 가공/조합/판매 가능 (하루 배분의 판단)

### 소비재 소모

- 물약, 음식, 폭탄 등 가져간 소비재는 사용 시 소진
- 준비 부족 → 탐사 중 위험 증가 → 조기 귀환 압박
- **소비재 = 조합 시스템의 산출물** → 탐사가 조합 수요를 만들어냄

### 유닛/플레이어 부상 리스크

- 전투 결과에 따라 유닛 및 플레이어 부상 가능
- **유닛 부상**: 회복까지 해당 유닛 탐사 불가 → 파티 약화
- **플레이어 부상**: 회복까지 탐사 자체 불가 → 가장 큰 리스크
  - 플레이어 부상 중에는 가공/조합/판매/육성만 가능
  - "다치면 며칠 동안 장사만 해야 한다" → 탐사 전 귀환 판단이 무거워짐
- 부상 회복: 시간 경과 or 치료 아이템 (조합 시스템과 연결)
- **부상 → 파티 약화 → 탐사 범위 축소 → 수입 감소** 연쇄
- 최악의 경우 유닛 상실 → 경영/관계에 큰 타격 (이머전트 이벤트)

## 탐사와 다른 시스템의 간섭

```
[탐사 → 조합]
  - 재료 공급: 탐사가 유일한 원재료 유입 경로
  - 채집 도구 등급이 재료 tier를 결정
  - 특정 층에서만 나오는 재료 → 특수 레시피의 전제 조건

[탐사 → 경영]
  - 탐사 비용(소비재, 스태미나)이 경영 지출
  - 탐사에서 얻은 재료가 제작→판매의 원료
  - 탐사 실패(부상/소실)가 경영에 타격

[탐사 → 관계]
  - 유닛 동행 시 호감도 변동
  - 탐사 중 유닛 관련 이벤트 발생 (대사, 반응, 특수 행동)
  - 유닛 부상/상실 → 관계 이벤트 트리거
  - 유닛 육성 이력에 "동행 던전" 기록 → 특성 분화에 영향

[탐사 → 내러티브]
  - 던전 텍스트: 특정 층 도달 시 로어 파편 발견
  - 숨겨진 구역에서 파편 로어 출현
  - 탐사 마일스톤 달성 → 노드 분기 출현 조건
  - 보스 격파 → 마일스톤 + 새로운 이벤트 풀 해금

[탐사 ← 달력]
  - 시즌에 따라 특정 층의 채집 가능 재료 변화
  - 달력 이벤트로 특수 던전 구역 한시적 개방
```
