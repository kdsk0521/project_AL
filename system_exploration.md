# 탐사 시스템 설계 (v0.8)

> 상위 문서: game_blueprint.md (코어 시스템 1축: 탐사)

---

## 개요

탐사는 게임의 **재료 유입 파이프라인**이자 스태미나를 가장 많이 소모하는 행동.
던전에 들어가서 채집, 전투, 이벤트를 겪고 귀환하여 재료를 가져온다.

```
탐사 준비 (편성, 장비, 소비재) → 던전 진입 → 층/구역 탐색 → 채집/전투/이벤트 → 귀환
       ↓                                                              ↓
  스태미나 소모                                              재료/로어/경험 획득
```

---

## 설계 원칙

> **"구조는 알지만, 안에서 무슨 일이 일어날지는 모른다."**

- 던전의 큰 틀(층, 구역, 보스)은 고정 → 공략의 재미, 계획 가능
- 내부(유닛 조우, 이벤트, 채집 포인트, 드랍)는 매번 랜덤 → 반복 탐사에 변화
- 탐사는 단순 파밍이 아닌 **비용과 위험을 수반하는 의사결정** (닼던 참고)
- 탐사 결과가 경영/관계/조합 전체에 파급 → 시스템 간섭의 핵심 공급원

---

## 던전 구조

### 구조 방식: 고정 골격 + 랜덤 내부 (세계수 × 이상한 던전)

#### 고정 요소 (매번 동일)

- **층수**: 1층, 2층, 3층... 무한 확장
- **구역 배치**: 각 층의 구역 구성 (광석 구역, 수풀 구역, 수계 구역 등)
- **보스 위치**: 특정 층(5층, 10층, 15층 등)마다 보스 배치
- **채집 가능 재료 종류**: "3층에 가면 철광석이 나온다"는 고정
- **재료 tier**: 깊은 층일수록 상위 tier 재료
- **환경 특성**: 층마다 속성/특성이 있음 (화산층, 수몰층, 결빙층 등)

#### 랜덤 요소 (매 진입 시 변동)

- **유닛 조우**: 어떤 유닛이 어디에 출현하는지 (적 = 유닛. 모든 조우 대상은 유닛)
- **채집 포인트 위치**: 어디서 캘 수 있는지
- **드랍 내용**: 같은 유닛/포인트여도 나오는 구체 아이템이 다름
- **이벤트**: 층 내 랜덤 이벤트 발생 (이머전트 이벤트 풀에서 추첨)
- **숨겨진 구역**: 일정 확률로 출현하는 숨겨진 방/통로

### 층 구조 예시 (초안)

```
[1~5층: 초급 구간]
  - 기초 재료 (tier 1)
  - 약한 유닛 조우
  - 5층: 첫 보스
  - 기본 채집 도구로 접근 가능

[6~15층: 중급 구간]
  - 중급 재료 (tier 2)
  - 환경 특성 등장 (화산, 수몰 등), 중급 유닛 조우
  - 10층, 15층: 보스
  - 상위 채집 도구 필요
  - 숨겨진 구역 출현 시작

[16~30층: 상급 구간]
  - 상급 재료 (tier 3)
  - 복합 환경 (화산+독기 등)
  - 특수 이벤트 빈도 증가
  - 상급/희귀 유닛 조우 가능

[31층~: 심층]
  - 최상위 재료 (tier 4+)
  - 노드 분기 관련 구역 출현
  - 파편 로어 밀도 최대
  - 무한 확장 (층수 스케일링)
```

### 무한 확장 구조

- 층수에 상한 없음. 깊이 갈수록 난이도/보상이 스케일링
- 특정 층마다 마일스톤 (최초 돌파 시 해금 보상)
- 일정 구간마다 새로운 환경 특성, 재료 tier, 이벤트 풀 추가
- **"아직 안 가본 층이 있다"가 탐사 동기**

## 탐사 흐름

### 1. 탐사 준비

```
- 파티 편성: 플레이어 캐릭터(필수) + 동행 유닛 선택
- 장비 확인: 플레이어 및 유닛 장비 상태
- 소비재 준비: 물약, 음식, 폭탄 등 (인벤토리에서 가져감)
- 도구 확인: 채집 도구 등급 → 채집 가능 재료 범위 결정
- 목표층 선택: 어디까지 갈 것인가 (이전 도달 최고층까지 직행 가능? — 미결정)
```

### 2. 던전 내 행동

층에 진입하면 탐색 시작. 각 층에서 가능한 행동:

- **이동**: 다음 구역/층으로 이동
- **채집**: 채집 포인트에서 재료 획득 (도구 등급에 따라 가용 범위)
- **조우**: 유닛 조우 시 대화/전투/도주 선택 (→ 조우 시스템 참조)
- **이벤트**: 랜덤/조건부 이벤트 발생 시 대응
- **탐색**: 숨겨진 구역/아이템 발견 시도
- **귀환**: 현재 시점에서 던전을 나감

### 3. 귀환 판단

탐사의 핵심 의사결정 중 하나.

> **"더 깊이 가면 더 좋은 재료가 있지만, 유닛이 다치고 소비재가 줄고 있다."**

- 언제든 귀환 가능 (귀환 아이템/스킬 필요 여부 — 미결정)
- 전멸 시 강제 귀환 + 채집물 일부 소실? (미결정)
- 귀환 시 채집한 재료, 발견한 로어, 획득한 경험이 확정
- 유닛 부상은 귀환 후에도 유지 → 경영/관계에 파급

## 조우 시스템

> **"던전에서 마주치는 모든 존재는 유닛이다. 적도, 동료 후보도, 같은 존재."**

### 핵심 원칙: 적 = 유닛

던전 내에서 조우하는 모든 존재는 **유닛**(system_unit.md 정의)이다.
포켓몬/메가텐/언더테일과 같은 구조 — 전투 상대이자 영입 대상이자 관계 대상.

- 던전 층마다 조우 가능 유닛 풀이 정의됨 (층별 고정, 구체 배치는 랜덤)
- 같은 유닛이라도 조우 때마다 미세한 개체차 존재 가능
- 보스도 유닛 (특수 유닛 — 영입/포획 가능 여부는 별도 규칙)

### 조우 시 선택지: 대화 / 전투 / 도주

유닛 조우 시 플레이어에게 3가지 기본 선택지가 주어진다.

```
[유닛 조우 발생]
       ↓
  ┌─────────┬─────────┬─────────┐
  │  대화   │  전투   │  도주   │
  └────┬────┴────┬────┴────┬────┘
       ↓         ↓         ↓
  메가텐식    자동전투    조우 회피
  교섭/영입   (기존)     (성공률 판정)
```

### 대화 (교섭/영입)

메가텐식 교섭. 유닛과 대화하여 영입을 시도한다.

- **준비가 핵심**: 탐사 전 준비한 아이템이 교섭의 도구가 됨
  - 음식, 선물, 특수 소비재 등을 제시하여 호감 획득
  - 유닛마다 선호/기피 아이템이 다름 (유닛 개별 설정)
- **특수 영입**: 특정 유닛은 특정 조건/아이템을 요구
  - "○○를 가져오면 함께 가겠다" 식의 퀘스트형 영입
  - 스토리/노드와 연결될 수 있음
- **교섭 실패 시**: 2단계 교섭 구조
  - **1차 실패**: 거절 (유닛 성격에 맞는 텍스트) → 플레이어 선택지 유지: 재시도 / 전투 / 도주
  - **2차 실패**: 성격 트레잇에 따라 유닛 반응이 갈림
    - 공격적 → 전투 전환 (강제 전투 돌입)
    - 소심/야생 → 도주 (유닛이 사라짐, 재조우 쿨타임)
    - 냉정 → 무시 ("더 이상 대화할 의사 없음", 이번 탐사에서 재교섭 불가)
    - 호기심 → 도주 (쿨타임 짧음)
  - 1차 실패에서 선택지가 살아있는 것이 핵심
  - 2차 실패 반응은 성격 트레잇 → 반응 타입 **1:1 매핑 테이블** (3종: 전투 전환 / 도주 / 무시)
- **영입 성공 시**: 해당 유닛이 플레이어 보유 유닛에 추가
  - 영입된 유닛은 즉시 파티 편입 가능 (슬롯 여유 시)

### 전투

→ 하단 "전투 시스템" 참조 (자동전투 + 최소 개입, AP 기반)

전투 경로를 선택하면 기존 자동전투 시스템으로 진입한다.

- **드랍 중심**: 전투 승리 시 해당 층에서 얻을 수 있는 재료 드랍
  - 드랍은 채집과 별도의 재료 유입 경로
  - 같은 층이라도 조우 유닛 종류에 따라 드랍 테이블이 다름
- **전투 승리 시 포획은 없음** → 드랍(재료)만
- **전투 중 관계 전환** (설계 의도 존재):
  - 전투 도중 특정 조건 충족 시 교섭/관계 시스템으로 전환 가능
  - "싸우다가 대화로 넘어간다" — ERA적 게임성의 핵심 경로
  - 조건 예시: HP 일정 이하, 특정 트레잇 보유, 특정 아이템 사용 등
  - → 상세 조건 및 전환 규칙은 미결정

### 도주

조우를 회피한다.

- 도주 성공률은 파티 상태, 유닛 특성 등에 따라 변동
- 도주 실패 시 강제 전투 진입 (선제 불이익 가능)
- 보스 조우 시 도주 불가 or 높은 패널티

### 유닛 포획/복제 (확정)

- **전투 승리 시 포획: 없음** → 드랍(재료)만 획득
- **미궁 복제체 개념: 폐기**
- 대화 = 영입, 전투 = 드랍(재료) 으로 역할 완전 분리 확정

### 전서(컴펜디엄) 구입 (확정)

- **도입 확정**
- 기초 풀 + 해금 확장 + 등록 후 재구매 — system_unit.md 전서 구매 구조 유지
- 재구매 가격 구체 수치는 밸런싱 단계 (미확정 유지)
- 탐사 영입과 전서 구입의 밸런스는 가격/상태 초기화로 조절

## 전투 시스템

### 방식: 자동전투 + 최소 개입 (백팩 히어로 참고)

> **"전투의 재미는 싸우는 도중이 아니라, 싸우기 전에 어떻게 세팅했느냐에 있다."**

전투 자체는 자동으로 진행된다. 유닛은 보유 트레잇(스킬)과 장비에 따라 자동 행동.
플레이어가 개입하는 지점은 **전투 전 준비**와 **전투 중 최소 판단** 두 곳뿐.

### 라운드 구조: 동시 해결 방식

- **라운드 동시 해결 방식 (오토배틀)** 확정
- 메가텐식 속도순 개별 행동이 아님
- 양측이 동시에 행동, 결과를 한꺼번에 처리

### 전투 스탯: 3종 (장비 의존)

- **물리공격 / 물리방어 / 속도** 3종
- 전부 **장비 의존** (유닛 자체는 종족 기본값 정도만 보유)
- 근접/사격 구분 없음 (물리 통합)

### 속도의 역할: AP(행동포인트) 시스템

- 속도 → **AP(행동포인트)로 변환** (라운드당 2~4 범위)
- 스킬마다 **AP 코스트** (1~3)
- 빠른 유닛 = 약한 행동 여러 번, 느린 유닛 = 강한 행동 한 번
- **부가 역할**: 속도차 → 회피율 보정 (별도 계산)

### 데미지 공식: 비율식

```
물리 데미지 = ATK × (ATK / (ATK + DEF_물리)) × 스킬배율
속성 데미지 = 스킬위력 × (스킬위력 / (스킬위력 + DEF_해당속성)) × 스킬배율
속성 부여   = 물리 데미지 + 부여속성 추가 데미지 (별도 계산)
```

- 비율식 채택 이유: 방어력이 아무리 올라도 데미지 0이 안 됨. 밸런싱 유연. 장비 격차가 체감되면서도 절벽이 없음
- 매직넘버/지수/로그 없이 단순한 구조 유지
- 스킬은 공식을 바꾸지 않고 **배율(×0.8~×1.5 등)로만 차별화**

### 방어 프로필: 6슬롯 수치 방식

```
유닛 방어 프로필:
  물리 방어  — 물리 공격 대응
  열 내성    — 열 속성 공격 대응
  위 내성    — 위 속성 공격 대응
  광 내성    — 광 속성 공격 대응
  동 내성    — 동 속성 공격 대응
  식 내성    — 식 속성 공격 대응
```

- **전부 수치 방식** (등급/약점 카테고리가 아님)
- 각 슬롯은 비율식 공식의 DEF에 직접 들어감
- 원소 간 상성/상극 없음 — 열 공격은 열 방어만 참조하는 단순 구조
- 수치 출처: 방어구의 **원소 태그 → 해당 속성 방어 수치**, **티어 → 수치 크기**
  - 조합 시스템이 방어 수치를 자동 생성하는 구조
- 장비로 방어 프로필 변경 가능 (약점 보완)

### 공격 속성 결정

- **물리 공격**: 무기 스탯 (항상 물리. 속성 무기는 만들지 않음)
- **속성 공격**: 스킬 트레잇으로만 (화염 마법 = 열, 지진 = 위 등)
- **속성 부여**: 소비재로 일시적 부여 (다크소울 송진 방식)
  - 물리 공격에 속성 추가 데미지가 붙음
  - 조합 시스템으로 부여 소비재 제작 → 기존 포션 체계와 연결

### 자동전투 AI (AP 체계)

기존 Dual Brain 세션 합의 내용 유지 + AP 체계에 얹음:

```
[라운드 시작]
유닛별 AP 확인
  ↓
1. 조건부 오버라이드 체크 (예: 심약 + HP<30% → 방어/철수 90%)
  → 해당 시 해당 행동 강제
  → 미해당 시 다음 단계
  ↓
2. 행동 선택 가중치 (성격 트레잇 기반)
  → 공격적: 공격 80%, 방어 10%, 스킬 10%
  → 소심: 방어 60%, 회피 20%, 공격 20%
  → etc.
  ↓
3. 선택된 행동 중 AP 맞는 스킬 선택
  ↓
4. AP 소진까지 반복
```

- **패시브 수치 보정** 별도 적용 (공격적: ATK+15%, DEF-10% 등)
- 구체 가중치/보정 수치는 밸런싱 단계

### 전투 전: 준비가 곧 전투

전투의 승패를 결정하는 것은 전투 전에 이미 정해져 있다.

- **파티 편성**: 누구를 데려가느냐 (트레잇 시너지, 방어 프로필 매칭)
- **장비 세팅**: 전투 수치는 장비에 의존 → 어떤 장비를 장착했느냐가 핵심
- **소비재 준비**: 물약, 음식, 폭탄 → 전투 중 플레이어가 사용할 자원
- **트레잇 조합**: 유닛 트레잇 간 시너지/상충 → 파티 전체의 전투 양상을 결정
- **교섭용 아이템**: 전투 중 관계 전환 시 사용할 소비재 (조우 시스템 연계)

### 전투 중: 유닛 자동 + 플레이어 최소 개입

```
[전투 시작]
       ↓
[매 라운드 동시 진행]
  - 양측 유닛: AP에 따라 자동 행동 (성격 트레잇 → AI 가중치)
  - 양측 행동 결과를 동시 해결
       ↓
[플레이어 개입 가능 (선택적)]
  - 소비재 사용: 물약, 폭탄 등 소모품 투입
  - 속성 부여: 소비재로 일시적 속성 추가 데미지
  - 철수 판단: 이 전투에서 빠진다 (도주)
  - 관계 전환 시도: 조건 충족 시 교섭으로 전환 (→ 조우 시스템 참조)
  - 그 외 전투 행동은 없음 — 유닛이 알아서 싸움
       ↓
[전투 종료]
  - 승리: 해당 층 드랍 획득, 경험치 배분
  - 관계 전환 성공: 교섭 → 영입/관계 진전
  - 패배: 유닛 기절, 강제 귀환 판정 가능
```

### 전투 텍스트 출력: 하이브리드 방식

- **기본**: 라운드별 요약 ("라운드 1: 아군 총 데미지 45, 적 총 데미지 30")
- **특이사항만 상세 출력**: 약점 히트, 기절, 전투불능, 합체 전환 시도 등

### 전투 결과를 결정하는 요소

| 요소 | 영향 | 플레이어가 제어하는 타이밍 |
|------|------|------------------------|
| 파티 편성 | 누가 싸우느냐 | 탐사 준비 |
| 장비 | 전투 수치 (공/방/속) + 방어 프로필 | 조합/제작 → 장착 |
| 스킬 트레잇 | 전투 중 자동 행동, AP 코스트 | 육성/합체 |
| 성격 트레잇 | AI 행동 패턴/가중치 + 조건부 오버라이드 | 육성 |
| 소비재 | 전투 중 유일한 개입 수단 (회복, 속성 부여) | 조합/제작 → 탐사 준비 |
| 방어 프로필 | 속성별 피해 경감 | 장비 세팅 (조합 시스템 연결) |

> **전투에서 직접 조작하는 건 거의 없지만, 전투 결과를 좌우하는 모든 것은 플레이어가 사전에 결정한 것.**

### 파티 시너지

- **알파 미포함, 후순위 추가 예정**
- 시스템 자리만 남겨두고 구현하지 않음

### 성격 트레잇과 전투 행동

유닛의 성격 트레잇이 자동전투 AI 가중치에 영향을 준다.
같은 스킬을 가져도 성격에 따라 다른 행동을 한다.

```
[공격적] → 공격 80%, 방어 10%, 스킬 10% / ATK+15%, DEF-10%
[소심]   → 방어 60%, 회피 20%, 공격 20% / HP<30%→방어/철수 90%
[냉정]   → 가장 효율적인 대상 선택, 상성 계산 / 패시브 보정 없음
[호기심] → 랜덤 행동 확률 증가, 예측 불가능
[순종적] → 안정적 패턴, 파티 협동기 발동률 증가
```

이것이 "같은 유닛이라도 어떻게 키웠느냐에 따라 전투에서도 다르게 행동한다"의 구현.

### 플레이어 캐릭터

- 탐사 파티의 **필수 멤버** (빠질 수 없음)
- 자체 스탯, 장비, 스킬 보유
- 전투 중에는 유닛과 마찬가지로 자동 행동
- 성장 가능 (레벨/경험치 — 상세 미결정)
- 유닛과의 차이점:
  - **납품 불가** (플레이어 자신이므로)
  - **상실 불가** (기절은 가능, 사망은 없음)
  - 합체 소재로 사용 불가
- 플레이어 기절 시:
  - 일정 기간 탐사 불가 → 그 동안 가공/조합/도시관리/육성만 가능
  - 치료 아이템으로 회복 단축 가능
  - **플레이어 기절 = 탐사 수입 완전 차단** → 가장 큰 리스크
- **전투에서 게임오버는 없다**: 전멸/플레이어 쓰러짐 시 강제 귀환 + 행동불가 페널티
  - 행동불가 기간(일 수)은 부상 정도에 따라 차등 (밸런싱)
  - 행동불가 중에도 고정 지출(세금, 유지비)은 계속 나감 → 실질적 경제 패널티
  - **게임오버는 경영 쪽(파산 엔딩 0-X)에서만 발생** → 진짜 리스크는 경영이 무너지는 것

### 전투 후: 기절과 탐사 계속 판단

#### HP 0 → 기절 (전투불능)

- HP 0이 되면 **기절** (전투불능)
- 기절 유닛은 해당 전투에서 행동 불가
- 전투 종료 후에도 기절 상태 유지
- 부상 "단계" (경상/중상) 구분 없음 — **기절 or 정상의 이분법**

#### 기절 유닛과 탐사 계속

- 기절 유닛을 **데리고 탐사 계속 가능**
- 디버프: 기절 유닛 1체당 **스태미나 추가 소모**
  - 이동, 전투, 채집 모든 행동에 추가 비용
  - 기절 유닛이 많을수록 스태미나가 빠르게 고갈
  - → 자연스러운 귀환 압박 ("더 갈 여유가 없다")
- 부담은 단계가 아니라 **기절 유닛 수 × 남은 탐사 깊이**로 표현

> **"더 깊이 가면 더 좋은 재료가 있지만, 기절 유닛이 늘고 스태미나가 빠르게 줄고 있다."**

#### 전멸 시

- **강제 귀환** (기존 확정 유지)
- 채집물 소실 → **탐사 경비 시설 등급에 따라 회수율 결정**
  - 시설 없음: 채집물 전부 소실
  - 시설 있음: 등급에 비례하여 회수
  - 배치 유닛 트레잇으로 회수율 보정

#### 귀환 후 회복

- **기본**: 시간 경과로 회복 (X일 — 구체 일수는 밸런싱)
- **단축**: 치료 아이템 사용 → 조합 시스템과 연결
- 회복 중 유닛: **파티 편성 불가 + 도시 배치 불가**
- 플레이어 기절 시: **탐사 자체 불가** (가장 큰 리스크, 기존 확정 유지)
- 회복 중에도 **고정 지출(세금, 유지비)은 계속** → 실질적 경제 패널티

### 탐사 경비 시설 (신규 도시 시설)

```
[탐사 경비 시설]
위치: 도시 시설 중 하나
역할:
  1. 전멸 시 유실물 회수 — 시설 등급 → 회수율
  2. 탐사 중 보급 — 시설 등급 → 보급 빈도/양

시설 등급에 따른 효과:
  없음  → 전멸 시 전부 소실, 보급 없음
  1등급 → 회수율 낮음, 보급 최소한
  2등급 → 회수율 중간, 보급 적정
  3등급 → 회수율 높음, 보급 풍부
  (구체 수치는 밸런싱 단계)

배치 유닛:
  트레잇에 따라 회수율/보급 효율 보정
  좋은 유닛 배치 → 탐사 안전 증가, 대신 파티에서 빠짐 (딜레마)

4축 간섭:
  경영(시설 투자) → 탐사(리스크 감소) → 조합(재료 보존) → 관계(배치 유닛 선택)
```

## 채집 시스템

### 핵심 구조

던전 내 채집 포인트에서 재료를 획득. **채집 도구 등급**이 접근 가능 범위를 결정.

```
[채집 판정]
채집 포인트 발견 → 도구 등급 확인
  → 도구 등급 충분: 채집 성공 → 재료 획득 (드랍 테이블에서)
  → 도구 등급 부족: "더 좋은 도구가 필요하다" → 채집 불가
```

### 채집 포인트 유형

| 유형 | 필요 도구 | 산출 재료 | 예시 |
|------|----------|----------|------|
| 광맥 | 곡괭이 | 광석류 | 철광석, 미스릴 원석 |
| 수풀 | 채집 낫 | 식물류 | 약초, 마나초, 독풀 |
| 수계 | 낚시대 | 수산/액체류 | 물고기, 마력수, 심층수 |
| 특수 | 전용 도구 | 희귀 재료 | 결정, 화석, 마석 등 |

- 같은 유형이라도 층에 따라 나오는 재료 tier가 다름
- 채집 시 스태미나 소모 (소량)

## 탐사 비용 구조

탐사는 **가장 스태미나를 많이 소모하는 행동**이자 소비재를 소진하는 행동.

### 스태미나 소모

```
탐사 1회 스태미나 = 기본 비용 + (목표 층수 × 층당 추가 비용)
```

- 깊은 층을 갈수록 스태미나 소모가 큼 → 하루에 깊은 탐사와 다른 행동을 병행하기 어려움
- 얕은 층 파밍은 스태미나 저비용 → 빠르게 기초 재료 확보 가능
- 탐사 후 남은 스태미나로 가공/조합/도시관리/육성 가능 (하루 배분의 판단)

### 소비재 소모

- 물약, 음식, 폭탄 등 가져간 소비재는 사용 시 소진
- 준비 부족 → 탐사 중 위험 증가 → 조기 귀환 압박
- **소비재 = 조합 시스템의 산출물** → 탐사가 조합 수요를 만들어냄

### 유닛/플레이어 부상 리스크

- 전투 결과에 따라 유닛 및 플레이어 부상 가능
- **유닛 부상**: 회복까지 해당 유닛 탐사 불가 → 파티 약화
- **플레이어 부상**: 회복까지 탐사 자체 불가 → 가장 큰 리스크
  - 플레이어 부상 중에는 가공/조합/도시관리/육성만 가능
  - "다치면 며칠 동안 장사만 해야 한다" → 탐사 전 귀환 판단이 무거워짐
- 부상 회복: 시간 경과 or 치료 아이템 (조합 시스템과 연결)
- **부상 → 파티 약화 → 탐사 범위 축소 → 수입 감소** 연쇄
- 최악의 경우 유닛 상실 → 경영/관계에 큰 타격 (이머전트 이벤트)

## 탐사와 다른 시스템의 간섭

```
[탐사 → 조합]
  - 재료 공급: 던전 전용 재료(약초/철광석/마력석)는 탐사만이 유일한 유입 경로
  - 기초 재료(물/슬라임핵/독물고기/촉매초)는 도시 시설에서도 수급 가능
  - 채집 도구 등급이 재료 tier를 결정
  - 특정 층에서만 나오는 재료 → 특수 레시피의 전제 조건

[탐사 → 경영]
  - 부족한 재료가 탐사 목표를 결정 ("약초가 없으니 던전에 가야 해")
  - 탐사에서 획득한 유닛이 도시 배치/납품/합체 자원
  - 탐사 실패(부상/소실)가 경영에 타격

[탐사 → 관계]
  - 유닛 동행 시 호감도 변동
  - 탐사 중 유닛 관련 이벤트 발생 (대사, 반응, 특수 행동)
  - 유닛 부상/상실 → 관계 이벤트 트리거
  - 유닛 육성 이력에 "동행 던전" 기록 → 특성 분화에 영향

[탐사 → 내러티브]
  - 던전 텍스트: 특정 층 도달 시 로어 파편 발견
  - 숨겨진 구역에서 파편 로어 출현
  - 탐사 마일스톤 달성 → 노드 분기 출현 조건
  - 보스 격파 → 마일스톤 + 새로운 이벤트 풀 해금

[탐사 ← 달력]
  - 시즌에 따라 특정 층의 채집 가능 재료 변화
  - 달력 이벤트로 특수 던전 구역 한시적 개방
```

### 확정됨 (이번 버전)
- [x] 유닛 조우 = 적 조우 통합 (적 = 유닛. 모든 조우 대상은 유닛)
- [x] 조우 시 기본 선택지: 대화 / 전투 / 도주
- [x] 대화 경로: 메가텐식 교섭. 준비 아이템이 교섭 도구
- [x] 전투 경로: 드랍 중심 (해당 층 재료). 기존 자동전투 시스템 사용
- [x] 전투 라운드 구조: 동시 해결 방식 (메가텐식 속도순이 아님)
- [x] 전투 스탯: 물리공격/물리방어/속도 3종, 장비 의존
- [x] 속도→AP 변환 시스템 (라운드당 2~4 AP, 스킬별 AP 코스트 1~3)
- [x] 데미지 공식: 비율식 (ATK×(ATK/(ATK+DEF))×스킬배율)
