# 조합/제작 시스템 설계 (v3.9)

> 상위 문서: game_blueprint.md (코어 시스템 2축: 조합/제작)
> 참조: world_setting.md (태그 체계, 재료 목록)

---

## 개요

이 게임의 조합/제작 시스템은 **4트랙**으로 구성된다.

| 트랙 | 방식 | 대상 | 상세 문서 |
|------|------|------|----------|
| 가공 | 원소 주입/해방에 의한 단일 재료 변환 | 원재료 → 조합용 소재 | 본 문서 |
| 일반 조합 | 특성 기반 자유 조합 | 소비재, 장비, 소재 | 본 문서 |
| 특수 레시피 | 고정 조합 (지정 재료 → 지정 결과) | 전설급, 스토리 아이템 | 본 문서 |
| 유닛 합체 | 종족×종족 테이블 (메가텐식) | 유닛 | 별도 문서 예정 |

본 문서는 **아이템 관련**(가공 + 일반 조합 + 특수 레시피)을 다룬다.

## 설계 원칙

> **"같은 재료라도 뭐랑 엮느냐에 따라 다른 결과가 나온다."**

- 고정 레시피 암기가 아닌, 재료 특성 조합에 의한 **판단**이 핵심
- 같은 재료여도 현재 영혼력, 유닛 상태, 도시 요청에 따라 최적 선택이 달라짐
- 조합 결과물에 파편 로어 텍스트를 부착하여 내러티브 레이어와 연결

> **"재료 수는 적게, 가공으로 깊이를 만든다."**

- 재료 종류가 많으면 "레시피 암기 게임", 적으면서 가공이 풍부하면 "판단의 게임"
- 플레이어가 전체 재료를 파악하고 있다는 느낌을 유지할 수 있는 범위
- 총 원재료 **12~16종**, 확장 포함 20종 이내

> **"억압이 있어서 자유가 있다. 자유도는 단계적으로 열린다."**

- **설비 추가 = 새로운 원소 가공 해금** → 진행 상태가 자유도의 범위를 결정
- 초기: 가마(열) + 분쇄기(위) 2종 → 후기: 5종 원소 가공 전부 개방
- 초보자는 제한된 범위에서 시스템을 익히고, 숙련자는 넓어진 범위에서 깊이를 누림

## 태그 체계 (확정)

모든 재료와 결과물은 **태그**로 정의된다. 태그는 3축 4분류.

### 기능 태그 — 효과 (10종)

결과물이 "뭘 하느냐"를 결정하는 태그.

| 구분 | 태그 | 방향 |
|------|------|------|
| 소비 | 회복 | 물약, 음식 (기존 회복/강화회복/영양 통합. 등급 차이는 tier/가공이 담당) |
| 소비 | 독성 | 독, 피해 아이템 |
| 소비 | 억제 | 봉인, 디버프 |
| 소비 | 분산 | 범위 효과 |
| 물리 | 공격 | 무기 계열 |
| 물리 | 수호 | 방어구 계열 |
| 기술 | 조련 | 육성/조교 도구 |
| 마법 | 부여 | 마법 장비, 장신구, 버프 |
| 변환 | 촉매 | 물리/화학적 변환, 가공 보조 |
| 변환 | 마력촉매 | 마법적 변환. 촉매의 마력 승격판 |

### 기능 태그 — 재료성질 (4종)

소재로서의 물성을 나타내는 태그.

| 태그 | 역할 |
|------|------|
| 경도 | 단단한 계열. 장비/건축 방향 |
| 가공성 | 가공 용이한 소재. 중간 소재/다용도/범용 베이스 |
| 전도 | 마력 전달. 마법 장비/도구 방향 |
| 보존 | 내구/유지. 보존 식량, 방어구 |

### 원소 태그 (5종)

마력의 에너지적 성질. **물리학의 에너지 유형을 마력으로 재해석**.

| 태그 | 에너지 기반 | 마력적 해석 |
|------|-----------|-----------|
| **열** | 열에너지 | 에너지가 변환되는 마력. 뜨거워지기도 차가워지기도 |
| **위** | 위치에너지 | 에너지가 축적된 마력. 눌려있다가 해방되는 |
| **광** | 빛에너지 | 에너지가 방출/전달되는 마력 |
| **동** | 운동에너지 | 에너지가 흐르고 이동하는 마력 |
| **식** | ★제5원소 | 마력이 물질의 본질 자체를 변질시키는 성질 |

#### 제5원소 "식"의 특수성

- 4원소는 에너지의 형태. 식은 **물질 자체의 변질** — 차원이 다른 현상
- 세계관 연결:
  - 심층에서 현실 법칙이 흐려지는 것 = 식의 작용
  - 차원 경계가 침식됨 → 방문자가 넘어오는 현상
  - 고대 문명이 심층에서 발견한/만난 것과 연결 (레이어 3 로어)
- 게임 내 운용:
  - Tier 1~2 재료에는 식 태그 거의 없음 (**예외: 약초** — 초반 복선)
  - 심층으로 갈수록 식 태그 재료 출현
  - 식 태그 재료 = 자동으로 "위험하고 특별한 것"

### 형태 태그 (7종+)

재료의 물리적 상태. **가공 전후로 변화하는 축**.

#### 원재료에 주로 붙는 형태

| 태그 | 채집 포인트 |
|------|-----------|
| 식물 | 수풀 채집 |
| 광물 | 광맥 채집 |
| 액체 | 수계 채집 |
| 유기 | 동물/마수 드랍 |

#### 가공/조합으로 생기는 형태

| 태그 | 파생 경로 |
|------|----------|
| 기체 | 증류, 연소 등의 가공 결과 |
| 분말 | 분쇄 가공 결과 |
| 결정 | 정제/농축 가공 결과 |

- 가공 체인에서 형태 태그가 자연스럽게 바뀜 (식물 → 건조 → 분말)
- **확장 가능**: 필요 시 잉곳, 직물, 합금 등 추가

### 태그 체계 총괄

| 축 | 종수 | 목록 |
|----|------|------|
| 기능 - 효과 | 10종 | 회복, 독성, 억제, 분산, 공격, 수호, 조련, 부여, 촉매, 마력촉매 |
| 기능 - 재료성질 | 4종 | 경도, 가공성, 전도, 보존 |
| 원소 | 5종 | 열, 위, 광, 동, 식(제5원소) |
| 형태 | 7종+ | 식물, 광물, 액체, 유기, 기체, 분말, 결정 (확장 가능) |

총 **26종+**, 축별 1~2개씩 부여되므로 조합 판정 관리 가능 범위.

## 원재료 (확정)

### 규모와 분포

재료 수는 적게, 가공으로 깊이를 만드는 설계.

| 구간 | 층 | Tier | 원재료 |
|------|-----|------|--------|
| 초급 | 1~5층 | 1 | **7종** (기초 재료. 태그 3축을 고루 커버) |
| 중급 | 6~15층 | 2 | 기존 재료 수급량 증가 + **신규 3~4종** |
| 상급 | 16~30층 | 3 | 기존 재료 계속 + **신규 2~3종** |
| 심층 | 31층~ | 4+ | **신규 1~2종** (심층 전용) |

- 총 원재료 **12~16종**, 확장 포함 20종 이내
- 층이 깊어져도 기존 재료가 계속 나옴 (수급량 증가)
- 신규 재료는 새 원소 태그나 기능 태그를 여는 역할에 한정
- **채집 포인트 ≠ 형태 고정**: 수풀에서도 유기가, 수계에서도 유기가 나올 수 있음

## 트랙 1: 가공 (원소 주입 체계)

### 핵심 구조: 가공 = 원소 주입/해방

가공은 단순한 물리 작업이 아니라 **재료에 특정 원소의 마력을 주입/해방하여 성질을 바꾸는 연금술 기술**.

각 가공 설비는 특정 원소에 대응하며, **설비 추가 = 새로운 원소를 다룰 수 있게 되는 것**.

### 가공 설비와 원소 대응 (쌍원소 구조)

모든 설비는 **주입 원소 + 해방 원소** 두 가지를 동시에 처리한다.

```
Tier 1:    가마 (열 주입 + 광 해방) + 분쇄기 (위 해방 + 식 해방)
Tier 1.5:  압축기 (위 주입 + 동 해방) — 동력 기관 제작으로 개방 (로어 힌트)
Tier 2+:   미결정 — 열 해방 / 광 주입 / 동 주입 / 식 주입 등 미개방 방향
```

- 초반 3설비로 5원소를 전부 한 번 이상 커버
- 식 주입이 가장 마지막 = 제5원소를 다루는 기술이 가장 위험하고 고급
- 고대 문명이 이걸 다루다가 멸망했을 가능성 (레이어 3 로어)

### 가공 원칙 (확정)

모든 설비는 **두 가지 원소 효과**를 동시에 발생시킨다 (쌍원소 가공).

- **주입 효과** → **기능 태그를 변환** (설비의 주된 가공 효과)
- **해방 효과** → **형태 태그를 변환** + 재료에 해당 원소가 있으면 **원소 태그 제거**

| 설비 | 물리적 행위 | 주입 원소 | 해방 원소 | Tier |
|------|-----------|----------|----------|------|
| 가마 | 굽는다 | 열 (기능 변환) | 광 (원소 제거) | 1 |
| 분쇄기 | 부순다 | 위 (기능 변환) | 식 (원소 제거) | 1 |
| 압축기 | 누른다 | 위 (기능 변환) | 동 (원소 제거) | 1.5 |

- **가마**: 열을 넣어 성질을 바꿈 → 부산물로 빛(광)이 빠져나감
- **분쇄기**: 축적(위)을 풀어 부숨 → 구조가 깨지며 기이함(식)도 빠짐
- **압축기**: 위치에너지(위)를 주입해 눌러 담음 → 운동(동)이 멈춤

### 가공 결과 태그 변환 (Tier 1 확정)

| # | 재료 | 기능 | 원소 | 형태 | 가마(열주입+광해방) | 분쇄(위해방+식해방) | 압축(위주입+동해방) |
|---|------|------|------|------|-------------------|-------------------|-------------------|
| 1 | 약초 | 회복 | 식 | 식물 | 기능: 회복→**보존** | 형태: 식물→**분말**, 원소: **식 제거** | 기능: 회복→**수호**, 형태: 식물→**결정** |
| 2 | 촉매초 | 촉매 | — | 식물 | 기능: 촉매→**억제** | 형태: 식물→**분말** | 기능: 촉매→**부여**, 형태: 식물→**결정** |
| 3 | 철광석 | 경도 | 위 | 광물 | 기능: 경도→**공격** | 형태: 광물→**분말** | 기능: 경도 유지, 형태: 광물→**결정** |
| 4 | 마력석 | 촉매 | 열 | 광물 | 기능: 촉매→**마력촉매** | 형태: 광물→**분말** | 기능: 촉매→**부여**, 형태: 광물→**결정** |
| 5 | 독물고기 | 독성 | 광 | 유기 | 기능: 독성→**가공성**, 원소: **광 제거** | 형태: 유기→**분말** | 기능: 독성→**조련**, 형태: 유기→**결정** |
| 6 | 물 | 가공성 | — | 액체 | 형태: 액체→**기체** (증기) | ✗ (분쇄 불가) | 기능: 가공성→**전도**, 형태: 액체→**결정** |
| 7 | 슬라임핵 | 전도 | 동 | 유기 | 기능: 전도→**수호** (경화) | 형태: 유기→**분말** | 기능: 전도→**보존**, 형태: 유기→**결정**, 원소: **동 제거** |

#### 설계 포인트

- **가마(열 주입)**: 7종 → 7종의 다른 기능 태그. 겹침 없음
- **분쇄(위 해방)**: 형태가 전부 분말. "부수면 가루" 한 문장으로 이해
- **압축(위 주입)**: 형태가 전부 결정. "누르면 굳는다" 한 문장으로 이해
- **쌍원소 가공**: 해방 원소와 일치하는 원소 태그가 재료에 있으면 제거됨
  - 가마(광 해방): 독물고기(광) → 광 제거 = 빛(마력 방출)이 빠져나가 해독
  - 분쇄(식 해방): 약초(식) → 식 제거 = 기이한 성질이 부서져 소멸
  - 압축(동 해방): 슬라임핵(동) → 동 제거 = 진동이 멈추며 에너지가 루프화
- **열+열 반응**: 마력석(열)에 가마(열 주입) → 촉매가 마력촉매로 승격
- **위+위 반응**: 철광석(위)에 분쇄(위 해방) → 축적된 것을 해방
- **원소가 없는 재료**: 촉매초(—), 물(—)은 해방 효과 해당 없음 → 기능/형태만 변환

### 가공 가능 여부

| 재료 | 가마(열) | 분쇄(위) | 압축(위) |
|------|---------|---------|---------|
| 약초 | ✓ | ✓ | ✓ |
| 촉매초 | ✓ | ✓ | ✓ |
| 철광석 | ✓ | ✓ | ✓ |
| 마력석 | ✓ | ✓ | ✓ |
| 독물고기 | ✓ | ✓ | ✓ |
| 물 | ✓ | ✗ (액체는 분쇄 불가) | ✓ |
| 슬라임핵 | ✓ | ✓ | ✓ |

### 가공 체인: 상한 없음 (자연 순환 자기 제한)

```
철광석 → [분쇄] → 철가루 → [가마] → 철 → [분쇄] → 철가루 → ...
```

- 가공만 반복하면 태그가 순환하여 새로운 결과가 안 나옴 → 플레이어가 알아서 멈춤
- 인위적 상한 불필요. 시스템이 자연스럽게 무한 이득 루트를 차단
- **진짜 의미 있는 체인**: 가공 사이에 조합을 끼워넣을 때 발생
  - 예: 철광석 → [분쇄] → 철가루 → [조합: +마력석] → 선철 → [가마] → 강철

### 가공의 역할 (요약)

- **기능 태그 변환**: 가마로 같은 재료를 다른 용도의 소재로 전환
- **형태 변환**: 분쇄로 고체를 분말로 → 조합 시 다른 반응
- **등급 상승**: 가공 체인(가공→조합→가공)을 거치며 tier 상승 가능
- **조합 공간 확장**: 재료 7종 × 가공 2종 = 사실상 소재 14종+의 효과
- **설비 연결**: 설비 업그레이드가 가공 가능 범위를 확장 → 경영 시스템과 연결

### 가공과 다른 시스템의 연결

```
[탐사] → 원재료 획득
  ↓
[가공] → 기능 태그 변환 / 형태 변환 (설비 필요 → 경영)
  ↓
[조합] → 변환된 태그 기반으로 결과물 결정
  ↓
[경영] → 도시 투자 / 설비 업그레이드 → 새로운 가공 경로 개방
```

## 트랙 2: 일반 조합 (특성 기반)

### 핵심 구조

재료에 **특성 태그**가 붙어 있고, 조합 시 태그 조합이 결과물의 **카테고리**와 **효과**를 결정한다.

```
재료 A (회복, 식, 식물) + 재료 B (가공성, 액체)
  → 태그 조합 판정: 회복 + 가공성
  → 결과: 기초 회복 물약

재료 A (회복, 식, 식물) + 재료 C (경도, 위, 광물)
  → 태그 조합 판정: 회복 + 경도
  → 결과: 회복 효과가 있는 장비? 또는 미지의 결과물
```

같은 약초라도 상대 재료에 따라 완전히 다른 결과물이 나온다.

### 결과물 결정 로직: 태그 합산 방식

> **"결과물은 매트릭스가 아니라 태그들의 합산으로 결정된다."**
> **"각 태그가 결과물에 자기 성질을 기여하고, 전체가 합쳐져서 결과물의 정체가 된다."**

레이어 분리(기능→원소→형태)가 아닌, **4축 전체가 동시에** 결과물을 정의한다.

- **기능(효과)** → "이 물건이 뭘 하는가" (목적)
- **기능(재료성질)** → "이 물건이 물리적으로 어떤가" (물성)
- **원소** → "어떤 에너지/방식으로 작동하는가" (작동 원리)
- **형태** → "물리적으로 뭔 모양인가" (형상)

```
예시: 조련(기능) + 전도(재료성질) + 동(원소) + 유기(형태)
 → 길들이는 + 에너지가 흐르는 + 진동하는 + 살아있는 느낌의 물건
 → 플레이어 네이밍: "전동딜도" (특수 레시피가 아니므로 플레이어가 이름을 붙임)

예시: 회복(기능) + 가공성(재료성질) + 식(원소) + 액체(형태)
 → 되돌리는 + 다루기 쉬운 + 본질변환 방식의 + 마실 수 있는 물건
 → "기초 회복 물약" (기초 레시피에 매칭되면 고정 이름)
```

### 태그별 효과 사전

각 태그가 결과물에 기여하는 성질. 이것이 조합 시스템의 기초 단위.

#### 기능 — 효과 태그 (10종): "뭘 하는 물건인가"

| 태그 | 결과물에 기여하는 성질 |
|------|---------------------|
| 회복 | 되돌린다, 채운다 |
| 독성 | 해친다, 침투한다 |
| 억제 | 막는다, 봉한다 |
| 분산 | 퍼뜨린다, 넓힌다 |
| 공격 | 부순다, 깨뜨린다 |
| 수호 | 지킨다, 버틴다 |
| 조련 | 길들인다, 변화시킨다 |
| 부여 | 덧붙인다, 올린다 |
| 촉매 | 변환한다 (거시/물리적) |
| 마력촉매 | 변환한다 (미시/화학적) |

- 촉매와 마력촉매는 상하 관계가 아니라 **같은 "변환" 축의 거시/미시 버전**
  - 촉매: 물리적 변환 — 형태를 재구성. 합치거나, 깨뜨리거나
  - 마력촉매: 화학적 변환 — 성질을 바꿈. 조성이 달라지거나, 반응하거나

#### 기능 — 재료성질 태그 (4종): "물리적으로 어떤 물건인가"

| 태그 | 결과물에 기여하는 성질 |
|------|---------------------|
| 경도 | 단단하다, 견고하다 |
| 가공성 | 다루기 쉽다, 섞이기 쉽다 |
| 전도 | 에너지/마력이 흐른다 |
| 보존 | 오래간다, 유지된다 |

- 효과 태그 = 목적을 준다 (이게 있으면 "쓸 수 있는 물건")
- 재료성질 태그 = 물성을 준다 (효과와 합쳐지면 완성품, 단독이면 소재/중간재)

#### 원소 태그 (5종): "어떤 방식으로 작동하는가"

> 물리학의 에너지 유형을 마력으로 재해석. 식만 "현실에 없는 것 전부"를 담당.
> 5원소 모두 같은 레벨 — 식이 특별히 상위이거나 분리되지 않음.

| 태그 | 물리학 기반 | 결과물에서의 의미 |
|------|-----------|----------------|
| 열 | 열에너지 | 온도, 에너지 변환. 뜨겁거나 차갑거나, 상태가 바뀌는 |
| 위 | 위치에너지 | 압축, 해방, 축적. 눌리거나 터지거나, 무거운 |
| 광 | 빛에너지 (전자 방출/흡수) | 방출, 흡수. 빛나거나 어둡거나, 보이거나 안 보이는 |
| 동 | 운동에너지 | 운동, 진동, 흐름. 움직이거나 흔들리거나 흐르는 |
| 식 | 현실에 없는 것 전부 | 본질 변환. 마법, 신성력, SF적 요소. "상처를 상처가 아니게" |

#### 형태 태그 (7종+): "물리적으로 뭔 모양인가"

| 태그 | 결과물에서의 의미 |
|------|----------------|
| 식물 | 유연한 것. 천, 섬유, 줄기, 잎 포함 |
| 광물 | 단단한 것. 장비류의 물리적 기반 |
| 액체 | 마시거나 뿌리는 것 |
| 유기 | 살아있는 것. 생물성 |
| 기체 | 퍼지는 것 |
| 분말 | 가루 |
| 결정 | 압축/응축된 것 |

### 조합 해석 규칙

#### 기본 원칙: 태그 합산

> **"재료를 넣으면 모든 태그가 합쳐진다. 합쳐진 태그 전체가 결과물의 정체."**

```
재료 A의 태그 전부 + 재료 B의 태그 전부 → 결과물의 태그 세트
```

- 각 태그가 독립적으로 효과를 기여
- 태그가 합쳐지면 **복합 동사**가 됨 ("되돌린다 + 부순다" = "부수고 되돌린다")
- 태그가 많을수록 복잡한 물건 (제작 난이도가 자연적 상한)
- 이름은 **플레이어가 붙인다** (특수 레시피만 고정 이름)
- **태그 중복은 세트 병합**: 같은 태그가 겹치면 1개로 합침 (수량/가중치 없음)
- **중복의 보상은 tier**: 중복 태그 수에 비례하여 결과물 tier 보정 ↑ (구체 수치는 밸런싱 시 확정)
  - 같은 성질의 재료를 겹쳐서 "순도를 높인다" 개념

#### 결과물 결정 흐름

```
1. 재료 A의 태그 + 재료 B의 태그 → 태그 조합 세트 생성
2. [특수 레시피 테이블 우선 확인] → 매칭되면 고유 아이템 생성 (고정 이름)
3. [일반 조합] 태그 합산 → 결과물의 정체를 태그 세트가 결정
   - 효과 태그 → 이 물건이 뭘 하는가
   - 재료성질 태그 → 물리적으로 어떤가
   - 원소 태그 → 어떤 방식으로 작동하는가
   - 형태 태그 → 물리적 형상
4. [모순 쌍 판정] → 태그 세트 내 활성 모순 쌍 수 카운트
   - 0~4쌍 → 해석 가능. 결과물 생성, 플레이어 네이밍
   - 5쌍 이상 → 해석 불가. 미지의 결과물 (태그 유지, 재가공/재조합 가능)
5. 재료 tier 평균으로 등급/수치 결정
```

#### 모순 쌍 판정 시스템 (해석 가능/불가능)

> **"태그 하나하나는 다 의미가 있다. 하지만 대립하는 태그가 너무 많이 공존하면, 하나의 물건으로 수렴하지 않는다."**

모순 쌍: 대립 관계에 있는 태그 2개가 동일 결과물에 동시에 존재하는 경우.

**모순 쌍 테이블 (총 12쌍)**

| 축 | 쌍 | 대립 논리 |
|----|-----|----------|
| 효과 | 회복 ↔ 공격 | 되돌린다 ↔ 부순다 |
| 효과 | 독성 ↔ 수호 | 해친다 ↔ 지킨다 |
| 효과 | 억제 ↔ 분산 | 막는다 ↔ 퍼뜨린다 |
| 효과 | 조련 ↔ 부여 | 길들인다 ↔ 덧붙인다 |
| 효과 | 촉매 ↔ 마력촉매 | 거시변환 ↔ 미시변환 |
| 재료성질 | 경도 ↔ 가공성 | 단단 ↔ 다루기쉬운 |
| 재료성질 | 전도 ↔ 보존 | 흐른다 ↔ 유지된다 |
| 원소 | 열 ↔ 광 | 에너지변환 ↔ 에너지방출 |
| 원소 | 위 ↔ 동 | 축적/해방 ↔ 흐름/이동 |
| 형태 | 기체 ↔ 결정 | 퍼지는 ↔ 압축된 |
| 형태 | 액체 ↔ 분말 | 흐르는 ↔ 부서진 고체 |
| 형태 | 식물 ↔ 광물 | 유연한 ↔ 단단한 |

**모순 판정에 안 잡히는 태그 (홀로 서는 태그)**
- **식** (원소): 비현실 에너지. 대립할 상대가 없음 — 오히려 "말이 안 되는 걸 말이 되게" 하는 쪽
- **유기** (형태): 살아있는 것. 뭐든 될 수 있음

**판정 기준**
- 활성 모순 쌍 **0~4개**: 해석 가능 → 복합 동사로 의미가 수렴
- 활성 모순 쌍 **5개 이상**: 해석 불가 → 미지의 결과물
- **특수 레시피 매칭 시**: 모순 쌍 수와 무관하게 고유 아이템 생성

```
예시: 모순 2쌍 — 해석 가능
약초(회복,식,식물) + 철광석(공격,위,광물)
→ 회복↔공격 (1), 식물↔광물 (1) = 모순 2쌍
→ "부수고 되돌리는 단단하고 유연한 물건" — 수술도구? 해석 가능

예시: 모순 5쌍 — 미지의 결과물
회복+공격, 독성+수호, 억제+분산, 경도+가공성, 열+광
→ "되돌리면서 부수면서 해치면서 지키면서 막으면서 퍼뜨리면서
   단단하면서 다루기쉽고 온도가 변하면서 빛나는"
→ 빌어먹을 미지의 무언가
```

#### 태그 조합 해석 가이드

**효과 + 효과 (복합 목적)**
- 두 효과가 합쳐져서 복합 동사가 됨
- 예: 회복+공격 = "부수고 되돌린다" (파괴적 재건, 수술)
- 예: 독성+수호 = "해치면서 지킨다" (독갑, 독가시)
- 예: 억제+분산 = "봉하면서 퍼뜨린다" (광역 봉인)
- 태그 4개 이상도 가능: 독성+회복+공격+수호 = "면역 물약" (침투해서 부수고, 되돌리고 지킨다)

**효과 + 재료성질 (목적 + 물성 = 완성품)**
- 자연스러운 완성품 조합. 효과가 목적을, 재료성질이 물성을 결정
- 예: 조련+전도 = "길들이는 + 에너지가 흐르는" (전도성 조련도구)
- 예: 회복+가공성 = "되돌리는 + 다루기 쉬운" (마시기 쉬운 회복물)

**재료성질 + 재료성질 (물성만 = 소재)**
- 목적(효과 태그)이 없으므로 "뭘 하는 물건"이 안 됨 → 소재/중간재
- 예: 경도+전도 = 단단하고 에너지가 흐르는 소재 (전도금속)
- 예: 가공성+보존 = 다루기 쉽고 오래가는 소재 → 식재료 방향

**원소의 역할 (작동 방식 결정)**
- 같은 기능 조합이라도 원소가 다르면 완전히 다른 물건
- 조련+전도+**동**+유기 = 진동하는 살아있는 조련도구
- 조련+전도+**열**+유기 = 온도가 변하는 살아있는 조련도구
- 조련+전도+**식**+유기 = 마법적으로 작용하는 살아있는 조련도구

**형태의 역할 (물리 형상 결정)**
- 회복+가공성+식+**액체** = 마실 수 있는 회복물 (물약)
- 회복+가공성+식+**분말** = 뿌리는/먹는 회복 가루
- 회복+가공성+식+**결정** = 응축된 회복 결정 (보석형)

### 특수 조합 테이블

특정 태그 조합에서 일반 합산과 다른 결과가 나오는 경우만 별도 정의.

- 특수 레시피 (기존): 지정 재료 → 고정 결과물, 고정 이름
- 특수 반응 (추후 확장): 특정 태그 조합에서 일어나는 창발적 효과
- → 특수 조합 테이블의 구체 내용은 미확정. 기본 시스템이 돌아간 후 필요에 따라 추가

### 네이밍 시스템

#### 핵심 구조

> **"시스템이 태그 기반으로 자동 생성명을 붙인다. 특수 레시피만 고정 이름."**

```
[정상 조합 — 해석 가능]
  자동 생성명 = 원소 접두사 + 재료성질 형용사 + 기능 키워드 + 형태 접미사
  예: "기이한 용이한 회복수" (식+가공성+회복+액체)

  태그가 적으면 짧고 (회복수), 많으면 길어짐 (자연스러운 복잡도 표현)
  티어 구분: 접두어 ("상급 ~", "정제된 ~") — 미확정

[특수 레시피]
  고정 고유명 ("기초 회복 물약", "심연의 물약")
  자동 생성명 대신 고유명 출력

[미지의 결과물 — 해석 불가]
  별도 네이밍 풀에서 뽑음: "일그러진 무언가", "이름 없는 잔해" 등

[리네임]
  최초 발견 시 1회 제안 → 수락하면 도감에 해당 이름으로 등록
  거절 시 자동 생성명 유지
  이후 도감에서 언제든 수정 가능
```

### 미지의 결과물

> **재료를 넣으면 반드시 무언가 나온다. 실패는 없다.**

- 모순 쌍 5개 이상 → **미지의 결과물**
- 미지의 결과물도 태그를 가짐 → 다시 가공/조합 재료가 됨
- 미지의 결과물 + 가공 = 정제: 불명확 특성이 정리되어 쓸 수 있는 소재로 구제
- 미지의 결과물에도 로어 텍스트 부착 가능

### 초반 조합 예시 (Tier 1 확정 재료 기반)

> 아래 이름은 예시일 뿐 — 실제로는 특수 레시피가 아닌 한 **플레이어가 이름을 붙인다**.

```
[기본 조합 — 태그 합산으로 읽기]
약초(회복,식,식물) + 물(가공성,액체)
 → 태그 합산: 회복+가공성+식+식물+액체
 → 되돌리는 + 다루기쉬운 + 본질변환 + 유연+마시는 → 회복 물약 계열

촉매초(촉매,식물) + 물(가공성,액체)
 → 태그 합산: 촉매+가공성+식물+액체
 → 변환하는(거시) + 다루기쉬운 + 유연+마시는 → 효과태그 없음 = 소재 (범용 액체)

[가공 후 조합]
보존허브(보존,식,식물) + 식재료(가공성,광,유기)
 → 보존+가공성+식+광+식물+유기 → 오래가는+다루기쉬운+빛에너지+살아있는 → 식량 계열

공격소재(공격,위,광물) + 슬라임핵(전도,동,유기)
 → 공격+전도+위+동+광물+유기 → 부수는+전도+축적+진동+단단+살아있는 → 마력 구동 무기 계열

수호소재(수호,동,유기) + 철광석(경도,위,광물)
 → 수호+경도+동+위+유기+광물 → 지키는+단단+흐르는+축적 → 방어구 계열

억제소재(억제,식물) + 전도분말(전도,동,분말)
 → 억제+전도+동+식물+분말 → 봉하는+전도+진동+유연+가루 → 마력 봉인탄 계열

마력촉매소재(마력촉매,열,광물) + 증기(가공성,기체)
 → 마력촉매+가공성+열+광물+기체 → 변환(미시)+다루기쉬운+에너지변환+퍼지는 → 상위 소재

독분말(독성,광,분말) + 촉매초(촉매,식물)
 → 독성+촉매+광+분말+식물 → 해치는+변환(거시)+방출+가루+유연 → 독 농축 소재
```

## 트랙 3: 특수 레시피 (고정 조합)

### 핵심 구조

특정 재료 조합에서만 나오는 고유 결과물. 일반 조합 규칙보다 **우선 판정**.

특수 레시피는 **기초 레시피**와 **상위 레시피** 2층으로 나뉜다.

### 기초 레시피 (안전망 — 튜토리얼 대체) (확정 9종)

게임 시작 시 전임자의 메모로 공개. 역할은 **시스템 작동 방식을 보여주는 것** + **초반 필수 아이템 확보**.
3단계로 나뉘어 가공 없이→가마→분쇄 순서로 시스템을 단계적으로 학습.

#### 1단계: 가공 없이 (원재료끼리)

| # | 이름 | 재료 | 태그 합산 | 용도 |
|---|------|------|----------|------|
| 1 | 기초 회복 물약 | 약초 + 물 | 회복+가공성+식+식물+액체 | 전투 중 HP 회복 |
| 2 | 기초 독물약 | 독물고기 + 물 | 독성+가공성+광+유기+액체 | 전투 중 투척 |
| 3 | 기초 배터리 | 마력석 + 슬라임핵 | 촉매+전도+열+동+광물+유기 | 조합 재료 (용도는 플레이어 탐색) |

- 약초+물 = 가장 먼저 배우는 조합. 조합 시스템 자체의 튜토리얼
- 독물약 = 전투 수단이 생김. "조합으로 탐사가 쉬워진다"를 체감
- 배터리 = **"조합 결과물이 다시 재료가 된다"를 가르치는 튜토리얼**. "이건 재료다, 다른 것과 합쳐봐라"까지만 안내. 구체 용도는 플레이어가 탐색

#### 2단계: 가마 가공 후 조합

| # | 이름 | 재료 | 가공 과정 | 태그 합산 | 용도 |
|---|------|------|----------|----------|------|
| 4 | 기초 무기 | 공격광물 + 슬라임핵 | 철광석→[가마]→공격광물 | 공격+전도+위+동+광물+유기 | 유닛 장착 (공격) |
| 5 | 기초 방어구 | 수호체 + 철광석 | 슬라임핵→[가마]→수호체 | 수호+경도+동+위+유기+광물 | 유닛 장착 (방어) |
| 6 | 기초 식량(스프) | 식재료 + 물 | 독물고기→[가마]→식재료 | 가공성+가공성+광+유기+액체 | 전투 이후 유닛 컨디션 |

- 무기/방어구: **같은 재료 2종(철+슬라임)의 가공 순서로 결과가 갈린다** — 가공 의미를 체감
- 식량: 독물고기를 구워서 해독 → 식재료화. "가공이 재료를 바꾼다"를 체감
- 전투 중(물약) vs 전투 이후(식량) 사용 타이밍 분리

#### 3단계: 분쇄 가공 후 조합

| # | 이름 | 재료 | 가공 과정 | 태그 합산 | 용도 |
|---|------|------|----------|----------|------|
| 7 | 기초 조교도구 | 슬라임분말 + 약초 | 슬라임핵→[분쇄]→슬라임분말 | 전도+회복+동+식+분말+식물 | 유닛 조교/육성 (ERA) |
| 8 | 기초 교섭재료 | 슬라임분말 + 촉매초 | 슬라임핵→[분쇄]→슬라임분말 | 전도+촉매+동+분말+식물 | 던전 유닛 교섭/영입 |
| 9 | 기초 로터 | 슬라임분말 + 슬라임핵 | 슬라임핵→[분쇄]→슬라임분말 | 전도+동+분말+유기 (중복태그 처리 미확정) | 유닛 조교 도구 (ERA) |

- 조교도구/교섭재료: **조련 태그가 Tier 1에 없어서 특수 레시피로만 가능** — "특수 레시피는 태그 규칙을 넘어서는 특별한 조합"임을 보여줌
- 교섭재료: 탐사 시스템의 조우→대화(교섭) 루트와 직접 연결 (메가텐식)
- 분쇄 설비 사용법 학습

#### 설계 포인트

- **Tier 1 원재료 7종 전부 사용**: 빠지는 재료 없음
- **3단계 튜토리얼**: 가공없이(1~3) → 가마(4~6) → 분쇄(7~9) 순서로 시스템 학습
- **전 카테고리 커버**: 회복/공격/장비/식량/조교/교섭/ERA도구/설비연료
- **대칭 구조**: 철+슬라임의 가공 순서로 무기/방어구 분기. 슬라임분말+α로 도구 3종 분기
- 채집 도구는 게임 시작 시 제공 (조합 불필요)
- **판매는 아이템이 아닌 유닛**: 아이템 판매 시스템 없음

### 상위 레시피 (발견형)

파편 로어, 마일스톤, 우연에 의해 발견되는 고유 레시피.

```json
{
  "recipe_id": "SPECIAL_001",
  "name": "심연의 물약",
  "tier": "special",
  "required_materials": ["MAT_ABYSS_METAL", "MAT_ORIGIN_WATER"],
  "known_from_start": false,
  "discoverable": true,
  "discovery_hint": "LORE_042",
  "unlock_condition": { "type": "milestone", "value": "dungeon_floor_50" },
  "lore": "심층의 금속과 심연의 물이 만나면, 경계를 넘는 자를 위한 약이 된다."
}
```

발견 방식:
- **힌트형**: 파편 로어에서 재료 조합 단서 → 플레이어가 유추
- **해금형**: 마일스톤 달성 시 레시피 공개
- **우연형**: 힌트 없이 시도해서 발견 (숨겨진 레시피)

### Tier 1 숨겨진 레시피 (15종 — 기초 레시피에 미포함, 플레이어 탐색)

기초 레시피로 알려주지 않지만 확실한 결과가 나오는 특수 레시피.
Tier 1 재료 + 가공 범위 내에서 만들 수 있는 것들.

#### 소비재 — 원재료 조합

| # | 이름 | 재료 | 태그 합산 | 용도 |
|---|------|------|----------|------|
| 1 | 해독제 | 약초 + 독물고기 | 회복+독성+식+광+식물+유기 | 독 상태 해제 |

#### 소재/중간재 — 원재료 조합

| # | 이름 | 재료 | 태그 합산 | 용도 |

#### 소비재 — 다단계 조합 (포션)

포션류는 **마력 용액(포션 베이스) + 가공된 재료** 2단계로 완성.

**용액 방향 가이드**:
- **마력 분말 용액** (촉매/물리적 변환) → **공격 계열** 포션에 사용. "물리적으로 형태를 부수고 재구성"
- **마력 촉매 용액** (마력촉매/화학적 변환) → **방어 계열** 포션에 사용. "화학적으로 성질을 바꿔 적응"
- 가이드일 뿐 강제 아님 — 반대로 넣어도 자동 생성으로 뭔가 나옴 (다른 방향의 결과물)
- 두 용액을 합치면? 촉매↔마력촉매 모순 쌍 1개 → 해석 가능하지만 불안정한 특수 소재 (자동 생성)

| # | 이름 | 재료 | 공정 | 용도 |
|---|------|------|------|------|
| 13 | 공격 강화 포션 | **마력 분말 용액** + 공격광물 | 마력석→[분쇄]→분말+물→용액, 철광석→[가마]→공격광물 | 아군 유닛 물리공격 ↑ |
| 14 | 방어 강화 포션 | **마력 촉매 용액** + 수호체 | 마력석→[가마]→촉매석+물→용액, 슬라임핵→[가마]→수호체 | 아군 유닛 물리방어 ↑ |
| 15 | 공격 약화 포션 | **마력 분말 용액** + 억제초 | 마력석→[분쇄]→분말+물→용액, 촉매초→[가마]→억제초 | 적 유닛 물리공격 ↓ (투척) |
| 16 | 방어 약화 포션 | **마력 촉매 용액** + 독물고기 | 마력석→[가마]→촉매석+물→용액 | 적 유닛 물리방어 ↓ (투척) |

- 가이드 방향과 다른 조합(분말용액+수호체, 촉매용액+공격광물 등)도 자동 생성으로 결과물 나옴 — 플레이어 탐색 영역
- 구체적 효과 차이는 밸런싱 시 확정

#### 소비재 — 다단계 조합 (술)

| # | 이름 | 재료 | 공정 | 용도 |
|---|------|------|------|------|
| 17 | 감로주 | 감로주 중간재 + 정화수 | 약초+촉매초 → 중간재, 촉매초+물 → 정화수 | 호감도/영입용. ERA 조교 시 보조 |
| 18 | 몽환주 | 몽환주 중간재 + 정화수 | 약초+억제초 → 중간재, 촉매초+물 → 정화수 | 조교 단계 하락용. ERA 핵심 도구 |

#### 설계 포인트

- 기초 레시피(9종)로 시스템을 배운 후, 플레이어가 **자발적으로 발견**하는 영역
- **소재 5종**: 마법강철(무기/장비), 마력용액2종(포션베이스), 가죽소재(방어구/천), 정화수(술베이스)
  - 조합 결과물이 다시 재료가 되는 패턴이 풍부함
- **포션은 2단계 공정**: 마력 용액 제작 → 용액+재료 = 포션. 용액 종류로 방향 분기 (상하 아님)
- **술은 3단계 공정**: 중간재 + 정화수. ERA 핵심 도구이므로 의도적으로 복잡
- 전투 스탯은 초반 **물리 공격/물리 방어/속도** 3종. 근접/사격 구분 없이 물리 통합
  - 속도 관련 포션은 상위 티어에서 추가
  - 원소 속성 공격/방어 포션은 전투 시스템 속성 상성 확정 후 추가 (미확정)

## 제작 판정 흐름 (통합)

### 핵심: 가공과 조합은 자유롭게 교차한다

```
가공 → 조합 → 가공 → 조합 → ... (필요한 만큼 반복)
```

#### 강철 제작 예시 (업데이트)

```
철광석 (경도, 위, 광물 / tier 1)
  → [분쇄] → 철가루 (경도, 위, 분말 / tier 1)
  → [조합: +마력석] → 선철 (경도+촉매, 위+열, 광물 / tier 2)
  → [가마] → 강철 (공격, 위+열, 광물 / tier 3)
```

#### 판정 흐름

```
[아이템을 작업대에 올린다]
       ↓
무엇을 할 것인가?
  ├─ [가공] → 설비 확인 → 가공 실행 → 기능태그 변환 or 형태 변환
  └─ [조합] → 두 번째 재료 투입
                 ↓
            [1단계] 특수 레시피 테이블 확인
                 → 매칭됨: 고유 아이템 생성
                 → 매칭 안 됨: 다음 단계로
                 ↓
            [2단계] 일반 조합 규칙 테이블 확인 (태그 기반)
                 → 매칭됨: 의도된 결과물 생성
                 → 매칭 안 됨: 다음 단계로
                 ↓
            [3단계] 미지의 결과물 생성 (실패 없음)

결과물을 다시 작업대에 올려서 추가 가공/조합 가능
```

### 실패 없는 조합

> **재료를 넣으면 반드시 무언가 나온다. 실패는 없다. 쓸모는 다를 뿐.**

- 미지의 결과물도 태그를 가짐 → 다시 가공/조합 재료가 됨
- 미지의 결과물 + 가공 = 정제: 불명확 특성이 정리되어 쓸 수 있는 소재로 구제
- 미지의 결과물에도 로어 텍스트 부착 가능

### 제작 매크로 (QoL — 향후 구현)

자주 쓰는 제작 루틴을 매크로로 저장하여 자동 실행.

- 한번 수동으로 성공한 루틴만 등록 가능
- 재료/설비 부족 시 중단 + 중단 지점 표시

*v3.0 — 결과물 결정 로직 전면 재설계. 3단계 레이어링 → 태그 합산 방식. 태그별 효과 사전 확정 (효과10+재료성질4+원소5+형태7 전종). 촉매/마력촉매 거시·미시 변환 관계 정의. 원소 5종 동일 레벨 확정 (식=비현실 에너지). 조합 해석 규칙 확정 (합산→특수우선→미지). 플레이어 네이밍 원칙.*
*v3.1 — 해석 가능/불가능 판정 기준 확정: 모순 쌍 카운트 방식. 모순 쌍 12종 정의 (효과5+재료성질2+원소2+형태3). 식·유기는 홀로(모순 불참). 5쌍 이상=미지. 분해 시스템 폐기.*
*v3.2 — 네이밍 시스템 확정. 자동 생성명(원소접두+재료성질형용사+기능키워드+형태접미사). 특수 레시피=고정명, 미지=별도 네이밍 풀. 최초 발견 시 리네임 1회 제안+도감 수정 가능. 가칭 치환 테이블 26종+ 정의 (열변/압쇄/광변/진동/기이 + 효과10종 + 재료성질4종 + 형태접미사7종).*
*v3.3 — 기초 특수 레시피 9종 확정. 3단계 튜토리얼(가공없이→가마→분쇄). Tier 1 원재료 7종 전부 사용. 전 카테고리 커버(회복/공격/장비/식량/조교/교섭/ERA도구/배터리). 판매=유닛만(아이템 판매 없음). 태그 중복 처리 미확정 항목 추가.*
*v3.4 — 태그 중복 처리 확정: 세트 병합(중복 무시) + 중복 태그 수에 비례하여 tier 보정. 순도를 높이는 개념. 구체 보정량은 밸런싱 시 확정.*
*v3.5 — Tier 1 숨겨진 레시피 15종 확정. 전투 스탯 방향: 물리공격·물리방어·속도, 근접사격 미구분.*
*v3.6 — 숨겨진 레시피 18종으로 확장. 포션 구조 변경: 마력 용액(포션 베이스) 2종 추가 — 마력분말용액(물리변환)·마력촉매용액(화학변환), 상하가 아닌 방향 분기. 포션은 2단계 공정(용액+재료). 가죽소재(수호체+촉매초) 추가. 소재 총 5종.*
*v3.7 — 포션 방향 가이드 명시: 마력분말용액→공격 계열, 마력촉매용액→방어 계열. 강제 아닌 가이드(반대 조합도 자동 생성 결과 산출). 두 용액 합성 시 촉매↔마력촉매 모순 1쌍 발생 → 해석 가능하지만 불안정한 특수 소재.*
*v3.8 — 쌍원소 가공 구조 도입. 모든 설비는 주입+해방 두 원소를 동시에 처리: 가마(열주입+광해방), 분쇄기(위해방+식해방), 압축기(위주입+동해방). 주입=기능변환, 해방=형태변환+원소제거. "원소 불변" 원칙 폐기 → 해방 원소 일치 시 원소 태그 제거됨. 초반 3설비로 5원소 전부 커버.*
*v3.9 — 압축기(위주입+동해방) 가공 결과 확정. 기능변환: 회복→수호, 촉매→부여, 경도 유지, 독성→조련, 가공성→전도, 전도→보존. 형태 전부 결정("누르면 굳는다"). 동 해방으로 슬라임핵(동) 원소 제거. 가공 가능 여부 테이블 압축 컬럼 추가.*
