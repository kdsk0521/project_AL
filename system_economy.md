# 경영 시스템 설계 (v1.1)

> 상위 문서: game_blueprint.md (코어 시스템 3축: 경영)
> 참조: system_crafting.md (조합/가공), system_unit.md (유닛/관계), system_unit_sigil.md (인(印) 체계)

---

## 개요

경영 시스템은 **비어있는 미궁도시를 유닛으로 채워가는 도시 복원 시뮬레이션**.
아이템 판매는 없다. 수입원은 오직 **유닛 납품(전서 포인트)**이며,
조합한 물건은 유닛에게 주거나 탐사에 쓰는 **상호작용 수단**이다.

```
[탐사] → 재료 수집 + 유닛 획득
       ↓
[조합/가공] → 장비, 소비재, 도구 제작
       ↓
[육성/조교] → 유닛 가치 상승
       ↓
[선택] → 유닛을 전서에 납품 (영혼력 획득)
       → 유닛을 도시에 배치 (시설 가동, 장기 투자)
       → 유닛을 탐사/합체에 사용 (전력 유지)
       ↓
[도시 성장] → 기초 재료 자동 수급, 새 기능 해금, 아이템으로 유닛과 상호작용
       ↓
[공방 확장] → 더 깊은 던전, 더 넓은 제작 범위
       ...순환
```

---

## 설계 원칙

> **"경영의 핵심은 유닛을 어디에 쓸 것인가의 판단이다."**

- **아이템 판매 없음**. 조합한 물건은 팔지 않고 사용한다
- 수입원은 오직 **유닛 납품 = 전서 포인트(영혼력)**
- 유닛의 가치는 **합체 소재 가치 + 조교/육성 결과**로 결정
- 조합물은 유닛에게 주는 **투자**이자 탐사의 **소비재**
- NPC 제로: 도시의 모든 기능은 **배치된 유닛이 운영**
- 경영의 재미 = "누구를 키워서, 어디에 쓸 것인가"

## 통화: 전서 포인트 (영혼력)

### 개념

공방을 가동하고 도시를 운영하려면 **영혼력**이 필요하다.
유닛을 전서에 납품하면 해당 유닛의 영혼력이 전서 포인트로 전환된다.

- 미궁도시의 공방은 영혼력으로 구동되는 설비 (로어와 연결)
- 왜 영혼력인지, 왜 도시가 비어있는지는 세계관/로어에서 다룸

### 영혼력 획득

유일한 정규 수입원 = **유닛 전서 납품**.

```
영혼력 = 기본 레벨값 + 트레잇 가산 + 육성 이력 보너스 + 인(印) 보정
```

#### 인(印) 보정의 구성 (v1.1 추가 — system_unit_sigil.md 참조)

**1. 인 기본 배율 (고정, 소폭)**
- 대부분의 정인: ×1.0 (보정 없음)
- 특수인 (염/수은/유황): ×1.15~1.2 (희소성 반영)

**2. 월별 수요 보정 (달력 연동)**
- 매달 2~3개 인(印)의 수요가 상승
- 해당 인 납품 시 영혼력 추가 보정 (예: ×1.2)
- 플레이어가 "이번 달에 뭘 납품할까" 판단하는 근거
- 키워둔 유닛이 이번 달에 비싸면 "팔까?" = 감정적 판단 포인트

**3. 인-트레잇 일치 보너스**
- 유닛의 인에 어울리는 트레잇 → 추가 보정
  - 백양 인 + 전투 특화 트레잇 = "순수한 하소체" → 보너스
  - 백양 인 + 성인 특화 트레잇 = "변형된 하소체" → 보너스 없음
- 인에 맞게 키우면 납품가 상승
- 인을 거슬러 키우면 납품가는 낮지만 합체 소재로 독특한 가치

> 인 보정 구체 수치는 미확정.

- 잘 키운 유닛일수록 영혼력이 높음
- 합체를 거친 유닛은 소재 유닛의 이력이 반영되어 더 높을 수 있음
- 호감도가 높은 유닛을 납품하면? → 이머전트 이벤트 (감정적 비용)
- **"이 유닛을 납품할 것인가, 남길 것인가"가 매 순간의 판단** (ERA식)

### 영혼력 지출

| 지출 유형 | 설명 | 시작 시점 |
|----------|------|----------|
| 공방 유지비 | 설비 규모에 비례하는 고정 지출 | **Tier 1.5** (압축기 건설 시점부터) |
| 도시 시설 건설 | 새 시설 설치 비용 | Tier 1~ |
| 전서 유닛 구매 | 전서에서 유닛 재구매 | Tier 1~ |
| 설비 제작/업그레이드 | 공방 가공 설비 확장 | Tier 1~ |

- **Tier 1 초반**: 영혼력 부담 거의 없음. 자유롭게 시스템 학습
- **Tier 1.5(압축기)부터**: 공방 유지비 시작. 사전 경고 제공. "이제부터 비용이 든다"
- **이후**: 설비/도시가 커질수록 유지비 증가 → 계속 유닛을 키워서 납품해야 유지

## 도시 시스템: 유닛 배치와 시설

### 핵심: 비어있는 미궁도시를 유닛으로 채운다

플레이어는 미궁도시의 시설에 유닛을 배치하여 도시를 복원한다.
배치된 유닛이 시설을 운영하고, 시설이 게임 기능을 제공한다.

- NPC는 없다. **배치된 유닛이 곧 NPC 역할**
- 도시 시설은 전서 포인트로 건설/확장
- 유닛의 트레잇에 따라 시설 운영 효율이 달라짐

### 도시 시설 (초안)

#### 자원 수급 시설

도시에서 기초 재료 일부를 자동 수급. **던전 탐사의 부담을 줄이는 보조 역할**.

| 시설 | 수급 재료 | 비고 |
|------|----------|------|
| 우물 | 물 | 초기 시설. 물이 여기저기 쓰이므로 가장 먼저 |
| 슬라임 농장 | 슬라임핵 | 슬라임계 유닛 배치. 야한 이벤트 발생 가능 |
| 낚시터 | 독물고기 | 수계 재료 양식 |
| 온실 | 촉매초 | **약초는 수급 불가** (식 원소 = 미궁에서만 자생) |
| 주방/식당 | (가공 결과물) | 후반 시설. 음식 자동 제작 등 |

#### 던전 전용 재료 (도시 수급 불가)

| 재료 | 이유 |
|------|------|
| 약초 | 식 원소 — 미궁의 기이한 에너지가 깃든 식물. 인공 재배 불가 |
| 철광석 | 광물 자동 수급은 밸런스 파괴. 던전 탐사 동기 유지 |
| 마력석 | 광물 자동 수급은 밸런스 파괴. 핵심 조합 재료이므로 던전 필수 |

- 유닛과의 교류/선물로 광물을 **가끔** 얻을 수는 있음 (비정기, 소량)
- 부족한 재료가 있어서 던전에 간다 — 탐사 동기의 핵심

#### 기능 시설

| 시설 | 기능 | 비고 |
|------|------|------|
| 전서 관리소 | 유닛 매매 (납품/구매) | 핵심 경제 시설 |
| 가공소 | 유닛에게 가공 위임 (자동 가공) | 스태미나 절약 |
| — | (향후 확장 가능) | — |

### 도시 배치와 상호작용

배치된 유닛에게 **아이템을 줄 수 있다** — 이것이 조합 결과물의 주요 사용처.

| 아이템 유형 | 효과 |
|------------|------|
| 장비 | 유닛 성능 강화 (탐사 유닛 / 도시 배치 유닛 공통) |
| 물약/소비재 | 탐사 소비재 / 도시 유닛 회복·강화 |
| 음식 | 탐사 소비재 / 도시 유닛 선물 (호감도) |
| 술 | 유닛 교섭/호감도 도구. 관계 이벤트 트리거 |
| 조교 도구 | 육성 방향 분기 |

- 유닛에게 좋은 아이템을 주면 → 시설 효율 상승, 호감도 상승, 이벤트 발생
- 유닛이 답례로 재료/복합태그 물품/정보를 돌려줄 수 있음
- **물건을 파는 게 아니라, 물건이 유닛과의 상호작용 수단이 되는 구조**

### 도시 유닛의 요청 (주문서 대체)

기존 "손님 주문" 시스템은 폐지. 대신 **도시에 배치된 유닛이 요청을 발생**시킨다.

- 배치 유닛이 특정 아이템을 요청 (필요 태그/형태 조건)
- 조건 충족 아이템을 주면 보상 (시설 효율 증가, 호감도, 특수 보상)
- 요청 내용은 시설 유형, 유닛 트레잇, 시즌에 따라 달라짐
- → 기존 "손님이 뭘 원하는가"의 판단이 "내 유닛이 뭘 필요로 하는가"로 전환

## 설비/시설 시스템

경영의 성장감은 **공방 설비 확장 + 도시 시설 확장**의 이중 축.

### 공방 설비 (가공/제작용)

| 설비 유형 | 역할 | 예시 |
|----------|------|------|
| 가공 설비 | 가공 방식 해금 / 가공 범위 확장 | 가마, 분쇄기, 압축기 |
| 제작 설비 | 조합 가능 카테고리/등급 확장 | 기초 작업대 → 고급 작업대 |
| 저장 설비 | 인벤토리/창고 용량 | 상자 → 창고 → 대형 창고 |

### 설비 업그레이드 구조

- 설비는 **조합으로 제작** (제작 도구 카테고리)
- 설비 등급이 올라가면:
  - 새로운 가공 경로 개방
  - 상위 tier 아이템 제작 가능
- 설비 업그레이드 자체가 **마일스톤 트리거**가 될 수 있음
- **Tier 1.5 이상 설비부터 공방 유지비(영혼력) 발생**

## 자금(영혼력) 순환

### 수입원

- **유닛 전서 납품**: 유일한 정규 수입. 유닛 가치에 비례
- **도시 유닛 답례**: 비정기, 소량. 재료/물품 형태 (영혼력은 아님)
- **특수 거래**: 마일스톤/노드 분기에서 발생하는 특별 거래 기회

### 지출처

#### 고정 지출 (Tier 1.5부터)

- **공방 유지비**: 가동 중인 설비 규모에 비례. 영혼력으로 월말 차감
  - 설비를 늘릴수록 유지비 증가 → "지금 확장할 여유가 있나?" 판단
  - **Tier 1 초반에는 유지비 없음** — 학습 구간 보호

#### 선택 지출

- **도시 시설 건설**: 새 시설 설치 (영혼력)
- **전서 유닛 구매**: 유닛 재구매 (영혼력)
- **설비 제작/업그레이드**: 재료 + 영혼력
- **탐사 비용**: 소비 아이템, 장비 소모 (재료 비용, 영혼력 아님)

### 자금 흐름의 긴장감

- Tier 1.5 이후 **고정 지출이 존재하므로 가만히 있으면 영혼력이 줄어든다**
- 유닛을 키워서 납품해야 유지 → "누구를 키워서 팔 것인가"
- 도시에 배치하면 장기 투자지만 당장 영혼력은 안 들어옴
- 합체에 쓰면 더 좋은 유닛이 나오지만 소재가 소멸
- **4방향 선택의 긴장**: 납품(수입) vs 배치(투자) vs 합체(강화) vs 탐사(전력)
- 이 긴장감이 경영 시스템의 재미이자 다른 시스템과의 간섭 지점

## 유닛의 4가지 용도 (경영 관점)

하나의 유닛을 두고 항상 4방향의 선택이 존재한다.

| 용도 | 효과 | 대가 | 인(印)과의 관계 |
|------|------|------|---------------|
| **전서 납품** | 영혼력(전서 포인트) 획득 | 유닛 소멸 | 인-트레잇 일치 → 납품가↑, 월별 활성 인 → 수요 보정 |
| **도시 배치** | 시설 운영, 기초 재료 수급, 도시 성장 | 탐사/합체에 못 씀 | (원소 속성이 주로 담당) |
| **합체 소재** | 더 강한/특수한 유닛 생산 | 소재 유닛 2체 소멸 | 인이 합체 테이블 키, 특수인은 계승 특수 효과 |
| **탐사/전투** | 던전 공략, 재료 확보 | 부상 리스크, 시간 투자 | 인에 맞는 육성 → 빠르게 강해짐 |

- 잘 키운 유닛일수록 모든 방향에서 가치가 높음 → 선택이 어려워짐
- 호감도가 높은 유닛을 납품/합체하면 → 이머전트 이벤트 (감정적 판단)
- **"이 유닛을 어디에 쓸 것인가"가 게임 전체를 관통하는 핵심 판단**

## 시간 구조

### 기본 단위: 월 (프린세스 메이커 참고)

게임의 시간은 **월 단위**로 흐른다.
한 달 안에서 플레이어는 스태미나 총량 내에서 자유롭게 행동한다.

```
[월 시작]
  - 도시 유닛 요청 갱신
  - 달력 이벤트 확인 (고정 이벤트, 시즌 이벤트)
  - 영혼력/재고 상태 확인
       ↓
[매일 — 하루 단위 행동]
  - 하루 스태미나 내에서 행동 선택
  - 탐사, 가공, 조합, 육성, 도시 관리 등
  - 행동별 스태미나 소모량이 다름
  - 스태미나 소진 or 자발적 종료 → 하루 종료
  - 다음 날 스태미나 리셋
       ↓
[월 말 (한 달치 일수 경과)]
  - 고정 지출 차감 (공방 유지비 — Tier 1.5 이후)
  - 도시 시설 산출물 정산 (자동 수급 재료 획득)
  - 이벤트 판정 (이머전트 이벤트, 마일스톤 체크)
       ↓
[다음 월 시작] → 반복
```

### 스태미나

- 플레이어의 **1일 행동량**을 제한하는 자원
- 매일 리셋 (하루가 끝나면 다음 날 전량 회복)
- 행동별 소모량 차등:
  - 탐사: 높음 (던전 깊이에 따라 추가 소모)
  - 조합/가공: 중간
  - 육성/조교: 중간~높음
  - 도시 관리/교류: 낮음
- 하루치 스태미나로 한두 가지 큰 행동 or 여러 가지 작은 행동을 배분
- **매일의 선택이 곧 한 달의 결과**: "오늘은 탐사" "오늘은 제작" "오늘은 육성"
- 스태미나 총량 확장 방식 (미결정: 설비? 아이템? 레벨?)

### 달력 시스템

월 단위 시간 구조 위에 **달력**이 존재한다.
1년 = 12개월, 1개월 = 30일 (확정)

#### 고정 이벤트 (시즌 이벤트)

- **시즌별 도시 유닛 요청 변화**: 시기에 따라 유닛이 원하는 아이템이 달라짐
- **축제/행사**: 특수 이벤트, 한정 거래 기회, 보너스 보상
- **계절 던전**: 특정 시기에만 열리는 던전 구역이나 채집 가능 재료
- **연중 이벤트**: 기념일, 유닛 관련 이벤트 (생일 등 — 관계 시스템 연결)

#### 달력과 인(印) 연동 (v1.1 추가)

매달 **활성 인 2~3개**가 순환. 1:1 고정 대응이 아닌 느슨한 보정.

- 해당 인 유닛의 조교 반응 전체에 소폭 보정 (예: 전 항목 +0.1x)
- 해당 인 유닛 납품 시 영혼력 추가 보정
- 해당 인의 유닛이 던전에서 조우 확률 상승 (?)

> 월별 활성 인 순환 테이블, 구체 효과/수치는 미확정.
> 달력×인 연동 상세는 → system_unit_sigil.md 참조

#### 달력과 다른 시스템의 연결

```
[달력 → 경영] 시즌별 유닛 요청 변화 → "이번 달엔 뭘 만들어줘야 하나" 판단
[달력 → 경영] 월별 활성 인(印) → "이번 달에 비싸게 팔 수 있는 유닛은?" 판단
[달력 → 탐사] 계절 한정 재료/던전 → "이 시기에만 갈 수 있는 곳" 탐사 동기
[달력 → 관계] 유닛 이벤트 → 호감도 상승 기회
[달력 → 이머전트] 시즌 조건이 이벤트 트리거에 추가됨
```

## 경영과 다른 시스템의 간섭

```
[경영 → 탐사]
  - 부족한 재료(약초/철광석/마력석)가 탐사 목표를 결정
  - 탐사 장비/소비재는 조합으로 제작
  - 도시 시설이 기초 재료를 대체하면 → 더 깊은 층에 집중 가능

[경영 → 조합]
  - 설비 등급이 제작 가능 범위를 결정
  - 도시 유닛 요청이 "뭘 만들까"의 방향을 제시
  - 조합물 = 유닛에게 주는 투자 수단

[경영 → 유닛/관계]
  - 유닛 납품 = 영혼력 획득 + 관계 변동 (이머전트 이벤트)
  - 도시 배치 = 유닛의 새로운 역할 부여
  - 아이템 선물 = 호감도/효율 상승
  - 영혼력 부족 → 유닛 납품 압박 → 감정적 판단

[경영 ← 탐사]
  - 던전에서 얻은 재료가 조합의 원료
  - 던전에서 획득한 유닛이 도시/납품/합체 자원
  - 탐사 실패(부상/소실)가 경영에 타격

[경영 ← 유닛/관계]
  - 유닛 트레잇이 도시 시설 효율에 영향
  - 유닛 이탈 시 도시 기능/전력 감소
  - 배치 유닛의 답례가 보조 수입
```
