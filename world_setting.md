# 세계관 설정 (v0.5)

> 상위 문서: game_blueprint.md

---

## 개요

이 게임의 세계관은 **ERA 메가텐식 혼합 패러디** 구조를 기반으로 한다.
오리지널 세계를 골격으로 하되, 다양한 작품의 패러디 캐릭터가 자연스럽게 존재할 수 있는 설정을 채택한다.

**핵심 설계 원칙: NPC 없음**
이 게임에 비유닛 NPC는 존재하지 않는다. 세계의 모든 캐릭터성 있는 존재는 유닛이다.
기능적 역할(세금, 주문 게시, 상점)은 **시스템/UI**가 담당한다.

---

## 세계 전제

### 기본 구조: "층이 있는 세계"

이 세계는 **표층(지상)**과 **심층(지하/이면)**으로 나뉜다.

- **표층**: 사람들이 살고, 가게를 열고, 거래하는 일상의 세계
- **심층(미궁)**: 표층 아래에 존재하는 거대한 지하 구조물. 마력이 농축되어 있으며, 깊이 갈수록 현실의 법칙이 흐려진다
- 표층과 심층의 경계에 **"문"**이 존재하며, 이 문을 통해 미궁에 진입한다

### 미궁(던전)의 정체

미궁은 **자연 형성된 마력의 지층** + **고대 유적의 잔해**가 복합된 존재.

- **고정된 층 구조**: 미궁의 골격은 고대 문명이 남긴 유적. 층수, 구역 배치, 보스 위치는 변하지 않는다 (세계수의 미궁식)
- **변동하는 내부**: 마력이 살아 움직이면서 내부 배치(적, 채집 포인트, 이벤트)가 매 진입 시 바뀐다 (메멘토스식)
- **깊이 = 마력 농도 = 위험도**: 깊이 갈수록 마력이 짙어지며, 더 강한 존재가 서식하고, 더 희귀한 재료가 산출된다
- **미궁은 끝이 없다**: 고대 유적 아래에도 마력 지층이 이어지며, 탐사할수록 새로운 층이 드러난다 (무한 확장)

#### 미궁과 로어의 연결

- 고대 유적 부분에는 이전 문명의 흔적(벽화, 유물, 기록)이 파편 로어로 존재
- 마력 지층 부분에는 마력 자체가 만들어낸 기이한 현상/존재가 로어의 소재
- 깊이 갈수록 "이 세계가 왜 이런 구조인지"에 대한 단서가 나타난다

### 연금술의 위상

이 세계에서 연금술은 **마력을 다루는 기술 체계**이자 산업의 핵심.

- 재료의 마력적 특성(태그)을 읽고, 조합하여 새로운 물질/존재를 만드는 기술
- 미궁에서 채취한 재료가 연금술의 원료 → 미궁 탐사가 산업의 기반
- **가공** = 재료의 마력적 특성을 변환하는 과정
- **조합** = 특성이 다른 재료를 합쳐 새로운 것을 만드는 과정
- **합체** = 연금술의 극치. 살아있는 존재의 본질을 합성하여 새로운 존재를 탄생시킴
- 연금술사는 이 세계에서 **장인이자 상인이자 연구자** — 사회적으로 인정받는 전문직

## NPC 제로 설계

### 원칙: 모든 캐릭터는 유닛이다

> **"이 세계에서 만나는 모든 존재는 잠재적으로 당신의 동료가 되거나, 소재가 되거나, 상품이 될 수 있다."**

비유닛 NPC는 존재하지 않는다. 이것이 가져오는 설계적 장점:

- **시스템 순도**: 모든 캐릭터가 유닛 시스템 안에 있음 → 합체/조교/판매/관계 전부 적용
- **콘텐츠 효율**: NPC 따로, 유닛 따로 만들 필요 없음. 모든 캐릭터 제작이 곧 유닛 제작
- **긴장감**: "이 가게 주인도 합체 소재가 될 수 있다" — 모든 관계에 시스템적 무게
- **감정적 복잡성**: NPC는 안전한 존재(절대 잃지 않음)이지만, 유닛은 잃을 수 있는 존재 → 모든 관계가 "이 존재를 어떻게 할 것인가"의 판단 대상

### 기능 역할의 시스템 대체

NPC가 하던 기능적 역할은 **시스템/UI/시설**이 대신한다.

| 기존 NPC 역할 | 시스템 대체 방식 |
|--------------|----------------|
| 상점 주인 | **상점 게시판**: 기초 재료/도구 목록이 자동 갱신. "누군가가 가져다 놓은 것" |
| 길드 접수원 | **의뢰 게시판**: 손님 주문이 자동 게시됨. "미궁 도시의 게시판 시스템" |
| 세금 징수원 | **자동 차감**: 월말에 시스템이 자동으로 세금을 가져감. "도시의 규칙" |
| 유닛 거래상 | **전서 시스템**: UI로 구현된 유닛 전서(도감). "마력으로 기록된 전서" |
| 멘토/스승 | **없음**: 튜토리얼은 기초 레시피와 시스템 자체가 담당 |
| 라이벌/경쟁자 | **이벤트로 처리**: 이머전트 이벤트에서 "다른 연금술사의 유닛이 도전해온다" 등 |

### 손님의 처리

경영 시스템의 "손님"도 NPC가 아닌 **시스템이 생성하는 주문서**로 처리.

```
[기존 설계]
손님(NPC)이 찾아와서 → "이런 물약 만들어줘" → 납품

[NPC 제로 설계]
의뢰 게시판에 주문서가 올라옴 → "회복 물약 tier 2 이상, 기한 5일"
  - 주문서에 짧은 플레이버 텍스트: "먼 길을 떠나는 자가 든든한 약을 찾고 있다"
  - 누가 의뢰했는지는 불명 (또는 로어 텍스트로 힌트만)
  - 납품하면 보상이 자동 지급
```

- 주문서의 플레이버 텍스트가 파편 로어 역할도 겸할 수 있음
- "누가 이걸 원하는 거지?"가 세계관적 궁금증을 자연스럽게 유발
- 특수 주문은 마일스톤/노드 분기에 의해 해금 → 서사와 연결

### 유닛이 하는 사회적 역할

NPC가 없으므로, 세계의 사회적 기능은 다음과 같이 구현:

- **도시의 분위기**: 텍스트 묘사로 전달. "공방 거리에는 오늘도 연기가 피어오르고 있다"
- **정보 전달**: 게시판, 공지, 아이템 설명문, 유닛 대사로 전달
- **사회적 상호작용**: 플레이어가 보유한 유닛과의 관계가 전부
  - 유닛이 세계에 대해 말해주는 것이 곧 세계관 전달
  - 호감도가 오르면 유닛이 더 많은 세계 이야기를 해줌 → 파편 로어와 일치
- **경쟁/대립**: 이머전트 이벤트로 처리. "미궁에서 다른 탐사 그룹과 마주쳤다" 등

## 존재 체계 (유닛의 세계관적 근거)

### "이 세계에는 다양한 존재가 산다"

미궁에서 마력이 분출되는 세계인 만큼, **비인간 존재**가 자연스럽게 존재한다.
이 세계의 "사람"은 인간뿐 아니라 다양한 종족을 포괄하는 개념.

### 종족 대분류

| 대분류 | 설명 | 예시 | 세계관적 존재 이유 |
|--------|------|------|------------------|
| **요괴** | 마력이 자연 발생시킨 존재. 각자의 성질과 의지가 있음 | 임프, 슬라임, 구미호, 오니 | 미궁의 마력이 형태를 얻은 것 |
| **정령** | 특정 원소/개념에 깃든 존재. 원소적 성질이 강함 | 화염 정령, 물의 정령, 숲의 정령 | 세계의 마력 순환을 담당하는 자연 존재 |
| **인조** | 연금술로 만들어진 존재. 골렘, 호문쿨루스 등 | 골렘, 인형, 키메라 | 연금술사가 제작한 피조물 |
| **야수** | 마력에 영향받은 동물/마수. 본능적 행동 | 마수, 환수, 마력 변이 동물 | 미궁 주변에서 마력에 적응한 생물 |
| **이형** | 분류 불가능한 기이한 존재. 심층에서 출현 | ???, 형태 없는 것들 | 심층의 농축 마력이 만들어낸 미지의 존재 |
| **방문자** | 이 세계 밖에서 온 존재 ★패러디 슬롯 | (동방, 원작 등의 캐릭터) | 미궁 심층에서 차원의 경계가 희미해짐 → 다른 세계에서 넘어온 존재 |

#### "방문자" — 패러디 캐릭터의 세계관적 근거

> **"미궁이 충분히 깊어진 곳에서는 세계의 경계가 희미해진다."**

- 심층의 마력 농도가 극한에 이르면 차원의 벽이 얇아짐
- 그 틈을 통해 **다른 세계의 존재**가 넘어오는 일이 간혹 발생
- 이것이 동방 프로젝트, 기타 작품 캐릭터가 유닛으로 등장하는 세계관적 이유
- 방문자는 원래 세계의 능력/성격을 일부 유지하지만, 이 세계의 마력 법칙에 적응
- **방문자도 합체 소재가 될 수 있음** — 마력 법칙 하에서 본질이 재구성되는 것
- 방문자 관련 파편 로어는 원작에 대한 패러디적 재해석이 될 수 있음

## 플레이어

### 포지션: 미궁가 연금술사

플레이어는 **미궁 입구 근처에 공방을 연 신참 연금술사**.

- 미궁에서 재료를 캐고
- 공방에서 가공/조합하고
- 가게에서 팔고
- 유닛을 계약/육성/합체하며
- 공방을 키워나간다

#### 플레이어의 세계관적 설정

- **신참**: 공방을 우연히 얻어 시작. 전임자와의 개인적 관계 없음
- **혼자 시작**: 처음에는 유닛 없이 혼자. 초기 미궁 탐사에서 첫 유닛을 만남/획득
- **성장 서사**: 무명의 신참 → 유명한 연금술사. 경영 규모가 서사의 척도
- 이름/외형/성별은 플레이어 설정 가능 (텍스트 기반이므로 자유도 높음)

### 전임자 (스타듀밸리 할아버지 포지션)

> **전임자는 죽었다. 그건 확실하다. 문제는 "왜"인가다.**

#### 기본 설정

- **사망 확정**: 전임자는 이미 죽었으며, 이 사실은 게임 시작 시 알려짐
- **플레이어와 무관**: 혈연도, 사제도 아님. 공방이 비어 있어서(또는 싸게 나와서) 플레이어가 들어온 것
- **정체는 점진적으로 드러남**: 처음엔 "이전 주인이 죽었다더라" 정도만 알 수 있음

#### 전임자가 남긴 것들 (게임 시작 시 플레이어가 발견)

```
[공방에 남아있는 것들]
- 낡은 작업대와 기초 설비 → 초기 가공/조합 가능 범위
- 서랍 속 메모 몇 장 → 기초 레시피 (튜토리얼 대체)
  "약초를 물에 달이면 회복약이 된다. 기초 중의 기초."
- 벽에 걸린 채집 도구 → 초기 채집 도구
- 책장에 꽂힌 낡은 노트 → 파편 로어 첫 실마리
  "미궁 깊은 곳에서 이상한 것을 봤다. 다음에 다시 가봐야겠다."
  (이 한 줄이 레이어 3의 첫 번째 파편)
```

#### 전임자의 미스터리 (장기 로어 라인)

전임자에 대한 정보는 파편으로 흩어져 있으며, 게임 진행과 함께 모인다:

- **아이템 설명문**: 전임자가 만든 것으로 보이는 오래된 도구/아이템이 미궁에서 발견
- **던전 텍스트**: 깊은 층에서 전임자의 흔적 (벽에 새긴 메모, 남겨진 장비)
- **유닛 대사**: 오래된 유닛이 "예전에 여기 왔던 연금술사를 기억한다"고 언급
- **주문서/게시판**: 오래된 미완료 주문이 남아있음 → 전임자가 마지막에 뭘 하려 했는지

전임자의 죽음이 의미하는 것:
- **미궁의 위험성을 체감**: "전임자도 실력 있는 연금술사였는데 죽었다"
- **특수 레시피 힌트**: 전임자의 연구 노트가 상위 레시피의 단서
- **노드 분기 연결**: 전임자가 발견한 것/하려 한 것이 노드 분기의 조건과 연결
- **엔딩 조건**: 전임자의 진짜 사인을 밝히는 것이 특정 엔딩(9-X)의 조건일 수 있음

#### 전임자의 사인 (개발자 내부 설정 — 플레이어에게 직접 공개 안 됨)

미결정. 다만 방향성:
- 단순 사고사가 아닐 것 (그러면 미스터리가 안 됨)
- 미궁 심층에서 뭔가를 발견한 것과 관련
- 전임자가 알아낸 것이 게임의 핵심 비밀(레이어 3)과 연결
- "전임자의 발자취를 따라가다 보면 세계의 진실에 닿는다"

## 시간과 공간

### 무대: 미궁 도시 (가칭)

미궁 입구를 중심으로 발전한 도시.
연금술사, 탐사가, 다양한 종족이 공존하는 거점.

#### 장소 구조 (NPC 제로 버전)

모든 장소는 **시설 = UI 인터페이스**로 구현. "가는 곳"이 아니라 "쓰는 기능".

| 장소 | 기능 | 구현 방식 |
|------|------|----------|
| **공방** | 가공, 조합, 합체 | 제작 시스템 UI. 설비에 따라 가용 범위 |
| **가게** | 진열 판매, 주문 확인 | 진열 슬롯 UI + 의뢰 게시판 |
| **미궁 입구** | 탐사 출발 | 탐사 준비 UI → 던전 진입 |
| **거처** | 유닛 관리, 육성, 교류 | 유닛 목록 + 육성/교류 시스템 |
| **전서** | 유닛 구매/재구매 | 전서 도감 UI |
| **상점** | 기초 재료/도구 구입 | 자동 갱신 상품 목록 (NPC 없음) |
| **게시판** | 공지, 이벤트, 힌트 | 시스템 메시지 + 달력 이벤트 표시 |

#### 장소와 톤

NPC가 없어도 세계의 분위기는 **텍스트 묘사**로 전달.
각 장소에 접근할 때마다 짧은 환경 묘사가 출력된다.

```
[공방]
"작업대 위에 어제 만들다 만 혼합물이 아직 남아있다. 가마에서 약간의 온기가 느껴진다."

[미궁 입구]
"거대한 문 너머로 찬 바람이 불어온다. 오늘따라 마력의 흐름이 불안정해 보인다."

[가게]
"진열대에 먼지가 쌓이기 전에 뭔가 올려놓는 게 좋겠다. 의뢰 게시판에 새 주문이 몇 개 걸려있다."
```

- 이 묘사가 시즌, 날씨, 마일스톤 달성, 이벤트 상태에 따라 바뀜
- **유닛의 코멘트**도 함께 출력 가능: "〈유닛이름〉이(가) 가게를 둘러보며 중얼거린다."
  → 유닛이 세계의 분위기를 전달하는 채널이 됨

### 시간 흐름

- game_blueprint의 월 단위 + 일 단위 스태미나 구조를 따름
- 1년 = **12개월** (현실 대응, 직관적)
- 1개월 = **30일** (단순화)
- 시즌: 봄(1~3월), 여름(4~6월), 가을(7~9월), 겨울(10~12월)
- 시즌에 따라 미궁 환경, 손님 수요, 이벤트가 변화

## 마력과 특성 태그 체계

### 마력의 성질

이 세계의 모든 재료/존재는 **마력적 특성**을 가진다.
이 특성이 곧 게임 시스템의 **태그**.

### 태그 체계 (확정)

태그는 **기능 태그(효과+재료성질) + 원소 태그 + 형태 태그**의 3축 4분류로 구성.
총 **26종+**, 축별 1~2개씩 부여되므로 조합 판정 관리 가능 범위.

#### 기능 태그 — 효과 (10종)

결과물이 "뭘 하느냐"를 결정하는 태그.

| 구분 | 태그 | 방향 |
|------|------|------|
| 소비 | 회복 | 물약, 음식 (등급 차이는 tier/가공이 담당) |
| 소비 | 독성 | 독, 피해 아이템 |
| 소비 | 억제 | 봉인, 디버프 |
| 소비 | 분산 | 범위 효과 |
| 물리 | 공격 | 무기 계열 |
| 물리 | 수호 | 방어구 계열 |
| 기술 | 조련 | 육성/조교 도구 |
| 마법 | 부여 | 마법 장비, 장신구, 버프 |
| 변환 | 촉매 | 물리/화학적 변환, 가공 보조 |
| 변환 | 마력촉매 | 마법적 변환. 촉매의 마력 승격판 |

#### 기능 태그 — 재료성질 (4종)

소재로서의 물성을 나타내는 태그.

| 태그 | 역할 |
|------|------|
| 경도 | 단단한 계열. 장비/건축 방향 |
| 가공성 | 가공 용이한 소재. 중간 소재/다용도/범용 베이스 |
| 전도 | 마력 전달. 마법 장비/도구 방향 |
| 보존 | 내구/유지. 보존 식량, 방어구 |

#### 원소 태그 (5종) — 마력의 에너지적 성질

물리학의 에너지 유형을 마력으로 재해석한 체계.

| 태그 | 에너지 기반 | 마력적 해석 |
|------|-----------|-----------|
| **열** | 열에너지 | 에너지가 변환되는 마력. 뜨거워지기도 차가워지기도 |
| **위** | 위치에너지 | 에너지가 축적된 마력. 눌려있다가 해방되는 |
| **광** | 빛에너지 | 에너지가 방출/전달되는 마력 |
| **동** | 운동에너지 | 에너지가 흐르고 이동하는 마력 |
| **식** | ★제5원소 | 마력이 물질의 본질 자체를 변질시키는 성질 |

##### 제5원소 "식"의 세계관적 의미

- 4원소는 에너지의 형태. 식은 **물질 자체의 변질** — 차원이 다른 현상
- 심층에서 현실 법칙이 흐려지는 것 = 식의 작용
- 차원 경계가 침식됨 → **방문자가 넘어오는 현상의 원인**
- 고대 문명이 심층에서 발견한/만난 것과 연결 (레이어 3 로어)
- 게임 내: Tier 1~2에는 식 태그 거의 없음 (**예외: 약초** — 초반 복선), 심층으로 갈수록 식 태그 재료 출현

#### 형태 태그 (7종+) — 가공 전후로 변화하는 축

| 구분 | 태그 | 설명 |
|------|------|------|
| 원재료 | 식물 | 수풀 채집 |
| 원재료 | 광물 | 광맥 채집 |
| 원재료 | 액체 | 수계 채집 |
| 원재료 | 유기 | 동물/마수 드랍 |
| 가공산물 | 기체 | 증류, 연소 등의 가공 결과 |
| 가공산물 | 분말 | 분쇄 가공 결과 |
| 가공산물 | 결정 | 정제/농축 가공 결과 |

- 가공 체인에서 형태 태그가 자연스럽게 바뀜 (식물 → 분말 등)
- **확장 가능**: 필요 시 잉곳, 직물, 합금 등 추가

#### 태그 체계 총괄

| 축 | 종수 | 목록 |
|----|------|------|
| 기능 - 효과 | 10종 | 회복, 독성, 억제, 분산, 공격, 수호, 조련, 부여, 촉매, 마력촉매 |
| 기능 - 재료성질 | 4종 | 경도, 가공성, 전도, 보존 |
| 원소 | 5종 | 열, 위, 광, 동, 식(제5원소) |
| 형태 | 7종+ | 식물, 광물, 액체, 유기, 기체, 분말, 결정 (확장 가능) |

### 태그 운용 방침

- 하나의 재료는 **기능 + 원소 + 형태** 조합으로 태그를 가짐
- 예: 약초 = [회복, 식, 식물] / 철광석 = [경도, 위, 광물] / 슬라임핵 = [전도, 동, 유기]
- **가공 = 원소 주입/해방**: 설비가 특정 원소에 대응, 기능 태그 변환 또는 형태 변환
- 조합 시 태그 조합이 결과물의 카테고리와 효과를 결정
- 원소 태그는 가공으로 안 바뀜 (재료 고유의 마력 성질)
- 태그 체계는 확장 가능 — 게임 범위 확대 시 새 태그 추가

## 재료 체계 (확정)

### 설계 원칙

> **"재료 수는 적게, 가공으로 깊이를 만든다."**

- 재료 종류가 많으면 "레시피 암기 게임", 적으면서 가공이 풍부하면 "판단의 게임"
- 플레이어가 전체 재료를 파악하고 있다는 느낌을 유지할 수 있는 범위
- **채집 포인트 ≠ 형태 고정**: 수풀에서도 유기가, 수계에서도 유기가 나올 수 있음

### 미궁 층별 재료 분포

| 구간 | 층 | Tier | 원재료 |
|------|-----|------|--------|
| 초급 | 1~5층 | 1 | **7종** (기초 재료. 태그 3축을 고루 커버) |
| 중급 | 6~15층 | 2 | 기존 재료 수급량 증가 + **신규 3~4종** |
| 상급 | 16~30층 | 3 | 기존 재료 계속 + **신규 2~3종** |
| 심층 | 31층~ | 4+ | **신규 1~2종** (심층 전용) |

- 총 원재료 **12~16종**, 확장 포함 20종 이내
- 층이 깊어져도 기존 재료가 계속 나옴 (수급량 증가)
- 신규 재료는 새 원소 태그나 기능 태그를 여는 역할에 한정

## 세계의 역사 (파편 로어의 뼈대)

> **이 역사는 플레이어에게 직접 전달되지 않는다.**
> 아이템 설명문, 던전 텍스트, 유닛 대사에 파편으로 흩어져 있으며,
> 플레이어가 스스로 조립해야 한다.

### 역사 레이어 (프롬소프트웨어식)

게임 내 역사는 3개 레이어로 구성.
**이것은 로어 콘텐츠의 내부 분류 체계이지, 출현 위치를 제한하는 것이 아니다.**

#### 레이어 1: 현재 — 일상과 표면

- 미궁 도시는 미궁 입구를 중심으로 자연 발생한 도시
- 연금술이 산업의 중심. 미궁 재료가 경제의 핵심
- 다양한 종족이 공존하며, 유닛 거래가 일상화
- "누가 미궁을 만들었는가"에 대한 답은 아무도 모름 — 그냥 있었다
- 도시에는 자치적 규칙이 있지만 명확한 통치자는 묘사되지 않음
  - 세금, 규칙은 "도시의 법"으로만 언급 → 시스템이 처리

#### 레이어 2: 고대 — 이전 문명의 흔적

- 미궁의 상부(1~30층)는 고대 문명이 건설한 구조물
- 이 문명은 연금술의 원형을 가지고 있었으며, 미궁을 특정 목적으로 건설
- 문명이 왜 멸망했는지는 파편으로만 암시
  - "그들은 더 깊이를 원했다"
  - "문을 열어서는 안 되는 곳의 문을 열었다"
  - "마지막 기록은 환희에 찬 문장으로 끝난다"
- 이 레이어의 로어가 **조합 특수 레시피의 힌트**와 연결

#### 레이어 3: 심층의 진실 — 세계의 근원

- 심층은 고대 문명도 다다르지 못한(또는 다다른 결과 멸망한) 영역
- 마력의 근원이 무엇인지, 미궁이 왜 끝없는지에 대한 단서
- "방문자"가 넘어오는 차원의 틈이 존재하는 이유
- 이 레이어의 로어는 **게임의 가장 깊은 비밀** — 노드 분기와 연결
- 진실의 전모를 아는 것 자체가 엔딩(9-X) 조건의 일부

### 핵심 원칙: 로어 출현 위치 ≠ 로어 레이어

> **"1층에서 주운 슬라임핵에 심층의 비밀이 적혀있을 수 있다."**

파편 로어는 **어디서든 어떤 레이어의 것이든 출현할 수 있다.**
레이어는 로어의 "주제 분류"이지, "출현 조건"이 아니다.

```
[출현 위치]          [나올 수 있는 로어 레이어]
1층 약초 설명문   →  레이어 3 ("이 풀의 뿌리는 심층까지 닿아있다고 한다")
5층 보스 드랍     →  레이어 2 ("고대의 문장이 새겨진 파편. 무엇을 봉인한 것일까")
유닛 기본 대사    →  레이어 3 ("...이 세계, 어딘가 이상하지 않아?")
초반 주문서 텍스트 →  레이어 2 ("옛 기록에 따르면 이 조합이...")
```

**차이는 "출현"이 아니라 "이해"에 있다:**

- 1층에서 심층 로어를 봐도 **그 시점에서는 뭔 소린지 모른다**
- 30층 도달하고 다른 파편을 모은 뒤에야 "아, 그때 그 약초 설명이..."
- **초반부터 호기심을 심어놓고, 후반에 연결되는 구조** = 프롬소프트웨어의 핵심

이것이 수집 동기를 극대화한다:
- 초반 플레이어: "이상한 문장이 있네, 뭔진 모르겠지만 기억해둬야지"
- 중반 플레이어: "아! 그때 그 설명이 이것과 연결되는 거였어!"
- 후반 플레이어: "처음부터 전부 깔려있었구나..."

### 로어 출현 밀도 가이드라인

레이어별 로어가 **출현할 확률**은 층과 무관하게 설계하되,
**밀도**는 아래와 같이 가이드라인을 둔다:

| 출현 위치 | 레이어 1 밀도 | 레이어 2 밀도 | 레이어 3 밀도 |
|----------|:---:|:---:|:---:|
| 아이템 설명문 (전 tier) | ●●● | ●● | ● |
| 던전 텍스트 (전 층) | ●●● | ●● | ● |
| 유닛 대사 (호감도 낮음) | ●●● | ● | — |
| 유닛 대사 (호감도 높음) | ● | ●●● | ●● |
| 이머전트 이벤트 | ●● | ●● | ●● |
| 노드 분기 | — | ●● | ●●● |

- 어디서든 모든 레이어가 나올 수 있지만, 비율은 조절
- 레이어 3은 흩어진 양이 적으므로 하나하나가 더 무겁게 느껴짐
- 유닛 호감도가 높을수록 깊은 이야기를 함 → 관계 시스템이 로어 발굴의 핵심 채널

### 로어와 시스템의 연결

```
레이어 1 (현재)  → 기초 레시피, 초기 이벤트, 세계관 입문
레이어 2 (고대)  → 특수 레시피 힌트, 마일스톤 로어, 조합 심화
레이어 3 (심층)  → 노드 분기 조건, 엔딩 9-X 조건, 세계의 비밀
모든 레이어      → 아이템 설명, 유닛 대사, 이벤트 텍스트에 분산 배치
```

## 시즌 이벤트 (초안)

달력 시스템과 연동되는 연중 고정 이벤트.

### 봄 (1~3월)

| 월 | 이벤트 | 효과 |
|----|--------|------|
| 1월 | 새해 / 미궁 개방제 | 미궁 1~5층 재료 산출량 증가. 특별 주문 출현 |
| 2월 | 해빙기 | 미궁 수계 구역 변동. 한정 수계 재료 출현 |
| 3월 | 봄 시장 | 주문 빈도 증가, 보상 보너스. "봄 맞이 세일" |

### 여름 (4~6월)

| 월 | 이벤트 | 효과 |
|----|--------|------|
| 4월 | 마력 활성기 | 미궁 전체 마력 농도 상승. 적 강화 + 재료 품질 상승 |
| 5월 | 미궁제 | 특수 던전 구역 한시 개방. 축제 한정 주문 |
| 6월 | 무더위 | 화산 구역 확장. 화계 재료 증가, 빙계 재료 감소 |

### 가을 (7~9월)

| 월 | 이벤트 | 효과 |
|----|--------|------|
| 7월 | 수확기 | 식물계 재료 산출 증가. 음식 관련 주문 증가 |
| 8월 | 상인의 달 | 가게 판매 보너스. 특수 거래 기회 |
| 9월 | 마력 침체기 | 미궁 마력 약화. 적 약화되지만 희귀 재료 출현률 감소 |

### 겨울 (10~12월)

| 월 | 이벤트 | 효과 |
|----|--------|------|
| 10월 | 결빙 시작 | 결빙 구역 확장. 빙계 재료 증가 |
| 11월 | 연말 준비 | 고급 주문 증가. "겨울 선물" 수요 |
| 12월 | 연말 축제 | 특수 이벤트 체인 발생. 유닛 관계 이벤트 집중. 보너스 기간 |

### 시즌 이벤트의 설계 의도

- 시세 변동 없이도 **시기에 따라 최적 행동이 달라짐**
- "다음 달에 미궁제가 있으니까 지금 재료를 비축해두자" → 선행 계획의 재미
- 시즌 한정 재료 → 수집 동기 + 시즌 특화 조합
- 유닛 이벤트가 특정 시즌에 집중 → "12월에 이 유닛과의 이벤트가..."

## 패러디 운용 방침

### 원칙

> **"패러디는 세계관의 뼈대가 아니라, 세계관 위에 얹는 장식이다."**

- 오리지널 세계관이 단독으로 성립해야 함
- 패러디 캐릭터(방문자)는 **없어도 게임이 돌아가는** 추가 콘텐츠
- 있으면 팬서비스/유머/수집 동기가 되지만, 의존하지는 않음

### 패러디 캐릭터 구현 방식

```json
{
  "unit_id": "UNIT_VISITOR_001",
  "name": "홍마관의 메이드",
  "origin": "visitor",
  "race": "방문자",
  "subtype": "요괴",
  "base_traits": ["완벽주의", "시간조작", "냉정"],
  "lore": "미궁 심층에서 발견된 은발의 여성. 자신이 어딘가의 메이드였다고 주장하며, 시간을 다루는 능력이 있다.",
  "parody_source": "동방 프로젝트 (이자요이 사쿠야 모티브)",
  "integration_note": "직접적인 원작 명칭 사용 대신 패러디적 재해석으로 구현"
}
```

- 원작 이름을 직접 쓰지 않고 **모티브/패러디로 재해석**
- 원작의 특징적 능력/성격은 트레잇으로 반영
- 파편 로어에서 원작에 대한 패러디적 힌트를 제공 (알아보는 재미)
- **방문자도 합체/조교/판매 가능** — 게임 시스템상 다른 유닛과 동등

## 세계관이 해소하는 미결정 사항

기존 블루프린트에서 "세계관 확정 시 결정"으로 남겨둔 항목들의 해소:

| 미결정 사항 | 해소 |
|------------|------|
| 세계관/소재 | 마력이 있는 판타지 세계, 미궁 도시 기반 |
| 플레이어 캐릭터 설정 | 미궁가 신참 연금술사. 공방 운영자 |
| 유닛 종족/타입 체계 | 요괴/정령/인조/야수/이형/방문자 6대 분류 |
| 특성 태그 체계 | 효과10+재료성질4+원소5+형태7 (3축 4분류, 26종+) |
| 던전 생성 방식 | 고정 골격(세계수) + 랜덤 내부(메멘토스) |
| 층별 구역 구성 | 환경별 구역 배치 (석굴→수계→화산→결빙→독림→유적→심층) |
| 1년 = 몇 개월 | 12개월 (현실 대응) |
| 1개월 = 몇 일 | 30일 |
| NPC 구조 | NPC 제로. 모든 캐릭터 = 유닛. 기능은 시스템이 대체 |
| 시즌 이벤트 | 월별 고정 이벤트 체계 (초안) |

## 미결정 사항

- [ ] 도시 이름 / 세계 이름
- [ ] 고대 문명의 구체적 설정 (미궁을 만든 존재의 정체)
- [ ] 방문자(패러디) 캐릭터 초기 라인업
- [ ] 종족별 합체 테이블 구조 (6대 분류 × 6대 분류)
- [ ] 원소 간 상호작용 규칙 (열+위 중첩 효과, 식의 특수 반응 등)
- [ ] 세계관 내 "연금술의 한계" (무엇은 만들 수 없는가)
- [ ] 미궁 밖의 세계 (다른 도시, 국가 등의 존재 여부 — 또는 의도적으로 불명으로 남길지)
- [ ] 게임 시작 시 초기 유닛 획득 이벤트 구체화
- [ ] 전임자의 구체적 사인 (레이어 3과 연결되는 핵심 비밀)
- [ ] 시즌 이벤트별 구체 콘텐츠 (한정 재료, 특수 주문, 유닛 이벤트)
- [ ] 환경 묘사 텍스트 풀 (장소별, 시즌별, 상태별)
- [ ] Tier 2~4 재료 구체화
- [ ] 형태 태그 확장 (잉곳, 직물, 합금 등 필요 시)
